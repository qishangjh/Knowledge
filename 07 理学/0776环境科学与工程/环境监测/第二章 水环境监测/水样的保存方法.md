# 水样的保存方法

采集的水样按监测项目**标准分析方法**规定添加适量保存剂,标准分析方法中没有规定的,按(HJ 493—2009)  规定执行。添加保存剂的过程中,所用器具不可混用,避免交叉污染。

| 序号  | 测试项目          | 采样容器  | 保存方法及保存剂用量                 | 可保存时间             | 最少采样量 | 备注  |
| --- | ------------- | ----- | -------------------------- | ----------------- | ----- | --- |
| 1   | COD、TOC、氨氮    | G     | 用 $H_2SO_4$酸化,pH≤2;        | 7d                | 250   |     |
|     | COD、TOC、总磷、总氮 | P     | -20℃冷冻                     |                   |       |     |
| 2   | DO            | 溶解氧瓶  | 加入硫酸锰，碱性KI叠氮化钠溶液，现场固定      | 24h               | 500   |     |
| 3   | $Cl^{-}$      | P/G   | 1-5℃，避光                    | 30d               | 250   |     |
| 4   | 总磷            |       |                            |                   |       |     |
| 5   | 总氮            |       |                            |                   |       |     |
| 6   | 总氰化物          | P/G   | 加NaOH到pH≥9,1-5℃冷藏          | 7d，如果有硫化物存在，保存12h | 250   |     |
| 7   | 锰、镉、铬         | P 或 G | 1 L 水样中加浓 $HNO_3$ 10 ml 酸化 | 1 月               | 100   |     |
| 8   | 铅             | P/G   | $HNO_3$ 酸化，pH1~2           | 1月                | 100   |     |  
P 为聚乙烯瓶(桶),G 为硬质玻璃瓶
