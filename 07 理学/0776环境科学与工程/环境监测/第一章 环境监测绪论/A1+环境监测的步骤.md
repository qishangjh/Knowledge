# A1+环境监测的步骤

## 监测方案的基本步骤

地表水检测方案制定
1. **基础资料的收集**
	1. 水体的水文、气候、地址和地貌资料
	2. 水体沿岸城市分布、工业布局、污染源及其排污情况、城市给排水情况等
	3. 水体沿岸的资源现状和水体资源的用途
	4. 饮用水源分布和重点水源保护区；水体流域土地功能及近期使用计划等。
	5. 历年水质监测资料
2. **监测断面和采样点的设置**
3. **采样时间和采样频率的确定**  
	力求以最低的采样频次，取得最有时间代表性的样品，既要满足能反映水质状况的要求，又要切实可行  
4. **采样和监测技术的选择**
5. **结果表达、质量保证及实施计划**
