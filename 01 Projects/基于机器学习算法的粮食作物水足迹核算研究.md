## 👋 README

**项目名称**:: [[基于机器学习算法的粮食作物水足迹核算研究]]
**核心目标**:: 结合计算机科学（机器学习）与环境科学，完成从数据收集、模型构建到论文撰写的全过程。
**技术栈**:: Python ([Scikit-learn](https://scikit-learn.org/)/[Pandas](https://pandas.pydata.org/)), [Gemini API](https://aistudio.google.com/), LaTeX
**资源状态**:: 
- Gemini API: $300 赠金 (有效期90天，优先使用)
- 虚拟卡: 余额 $1 (备用)
- 笔记库: Obsidian (700+ 篇存量)

> [!abstract] 研究思路
> 利用机器学习算法（如随机森林、LSTM、XGBoost等）对粮食作物（如水稻、小麦）的生产数据、气象数据进行训练，建立水足迹预测与核算模型，分析时空演变规律及驱动因子。

## 🙋 Milestones & Tasks

### 📅 第一阶段：文献综述与开题 (Literature & Proposal)
- [ ] **文献检索与筛选**
    - [ ] 使用 `Connected Papers` 寻找 "Crop Water Footprint Machine Learning" 相关的 Seed Paper。
    - [ ] 使用 `Consensus` 提问：目前[[主流的水足迹预测 ML 算法]]有哪些？
- [ ] **文献深度阅读**
    - [ ] Zotero 抓取 30-50 篇核心论文。
    - [ ] 配置 `沉浸式翻译` + `Gemini API` 进行双语阅读。
    - [ ] 利用 `Zotero Integration` 将高亮笔记导入 Obsidian。
- [ ] **开题报告撰写**
    - [ ] 在 `Obsidian Canvas` 中画出技术路线图（数据流 -> 算法 -> 结果）。
    - [ ] **[AI 任务]** 让 Gemini 生成开题报告大纲（研究背景、意义、内容）。

### 📅 第二阶段：数据收集与处理 (Data & Preprocessing)
- [ ] **数据获取**
    - [ ] 收集气象数据（降水、温度、ET0等，来源：[中国气象数据网](http://data.cma.cn/) 或 [FAO](https://www.fao.org/aquastat/en/)）。
    - [ ] 收集作物产量与灌溉数据（来源：[国家统计局](https://data.stats.gov.cn/)）。
- [ ] **数据清洗 (Python)**
    - [ ] **[AI 任务]** 使用 Gemini 2.5 Flash 写 Python 脚本处理缺失值和异常值。
    - [ ] 特征工程（筛选对水足迹影响最大的因子）。

### 📅 第三阶段：模型构建与实验 (Modeling)
- [ ] **模型搭建**
    - [ ] 划分训练集与测试集。
    - [ ] 搭建基准模型（如 LR, SVR）。
    - [ ] 搭建进阶模型（如 Random Forest, XGBoost, LSTM）。
- [ ] **模型评估与优化**
    - [ ] 计算 RMSE, MAE, R² 指标。
    - [ ] **[AI 任务]** 将实验报错信息发给 Gemini debug。
    - [ ] 使用 `Matplotlib` / `Seaborn` 绘制结果对比图（要求 Gemini 提供绘图代码）。

### 📅 第四阶段：论文撰写与排版 (Writing)
- [ ] **论文正文写作**
    - [ ] 搭建 LaTeX 环境（推荐直接用 Overleaf）。
    - [ ] 下载并配置 **[吉林大学本科毕业论文LaTeX模板 (GitHub)](https://github.com/maxuewei2/JLUThesis2020)** 或 **[Overleaf 在线模板](https://www.overleaf.com/latex/templates/jlu-thesis-2020/rdnmpjmrtqvc)**。
    - [ ] **[AI 任务 - 重点]** 使用 Gemini 对中文草稿进行 **学术化润色 (Polishing)** 和 **降重**。
    - [ ] 插入 Zotero 导出的参考文献 (`.bib` 文件)。
- [ ] **查重与定稿**

### 📅 第五阶段：结题与答辩 (Defense)
- [ ] **PPT 制作**
    - [ ] 使用 `Gamma` 导入论文摘要生成 PPT 初稿。

- [ ] **演讲稿准备**
