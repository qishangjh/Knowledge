# PM2.5和PM10的国标测定方法
以下是关于PM2.5和PM10的国标测定方法，以及单机监测、在线监测和遥感监测的综合对比：

---

## **PM2.5和PM10的国标测定方法（GB 3095-2012 & HJ 618-2011）**
1. **方法名称**：重量法（滤膜采样-称重法）  
2. **原理**：  
   - 通过采样泵抽取空气，利用切割头（PM10或PM2.5）分离颗粒物，收集在滤膜上。  
   - 通过恒温恒湿条件下称量滤膜采样前后的质量差，计算浓度（μg/m³）。  
3. **关键设备**：  
   - 切割头（PM10或PM2.5）、滤膜（玻璃纤维或石英纤维）、恒温恒湿天平室。  
4. **特点**：  
   - **准确性高**，是国标基准方法；  
   - **耗时较长**（需24小时连续采样和实验室分析）；  
   - **人力成本高**（需专业人员操作和称重）。

---

## **单机监测、在线监测、遥感监测对比**
| **对比维度** | **单机监测**（便携式/手持式） | **在线监测**（固定站自动监测）  | **遥感监测**（卫星/激光雷达）  |
| -------- | ----------------- | ------------------ | ------------------ |
| **灵敏度**  | 中（适合短期/应急监测）      | 高（连续监测，数据精确）       | 低至中（大范围趋势分析）       |
| **设备成本** | 低（1万~10万元）        | 高（50万~200万元）       | 极高（卫星：亿级；激光雷达：百万级） |
| **人力成本** | 中（需现场操作和维护）       | 低（自动化，远程管理）        | 中至高（需专业数据处理团队）     |
| **应用场景** | 移动监测、应急监测、室内检测    | 城市空气质量站、工业区长期监控    | 区域污染分布、跨省传输分析      |
| **数据频率** | 分钟级~小时级           | 实时（分钟级）            | 小时级~天级（卫星）         |
| **典型技术** | 光散射法、β射线法         | β射线法、微量振荡天平法（TEOM） | 激光雷达（Lidar）、卫星反演   |

---

## **补充说明**
1. **单机监测**：  
   - **优点**：灵活便携，适合临时布点或污染源排查（如工地扬尘监测）。  
   - **缺点**：受环境温湿度影响大，长期稳定性较差。  

2. **在线监测**：  
   - **优点**：数据连续、实时性强，支持污染预警和趋势分析。  
   - **缺点**：设备运维复杂，需定期校准（如HJ 653-2013规范）。  

3. **遥感监测**：  
   - **优点**：覆盖范围广，可监测垂直污染层和区域传输（如沙尘暴）。  
   - **缺点**：分辨率较低，无法精准定位点源污染。  

---

**实际应用建议**：  
- **城市空气质量网络**：以在线监测为主，单机监测补充应急需求。  
- **区域污染研究**：结合遥感监测（卫星/激光雷达）和地面在线数据。  
- **工业源监控**：在线监测+单机巡检结合，确保排放达标。
