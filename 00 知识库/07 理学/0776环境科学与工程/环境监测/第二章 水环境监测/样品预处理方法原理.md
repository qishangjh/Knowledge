# 样品预处理方法原理

1.列举样品预处理的原因和目的

2.列举消解的目的和方法（MOOC 2.9）

3.描述灰化法的基本操作和注意事项

| 方法  | 原理简述                        | 实例（目标混合物→产物）                         |
| --- | --------------------------- | ------------------------------------ |
| 过滤  | 利用多孔介质截留悬浮物，分离固液两相          | 河水悬浮物→澄清滤液（去除泥沙）                     |
| 离心  | 通过高速旋转产生的离心力分离密度差异组分        |                                      |
| 消解  | 使用强酸/氧化剂破坏有机物，转化金属为可溶态      | **含有机铬废水→Cr(VI)溶液**（硝酸+高锰酸消解测总铬）     |
| 灰化  | 高温灼烧（450-550°C）去除有机物，保留无机成分 | **土壤样品→重金属氧化物**（灰化后溶解测铅，不适用汞/砷）      |
| 气提  | 通入载气将挥发性物质从液相转移至气相          |                                      |
| 萃取  | 利用溶质在互不相溶两相中的分配差异进行分离       |                                      |
| 色谱  | 基于组分在固定相与流动相间的分配差异实现分离      | **有机氯农药混合物→单组分峰**（气相色谱-ECD分离六六六/DDT） |

**重点实例说明**：
1. **消解**：处理含铬电镀废水时，采用硫酸-磷酸-高锰酸钾多元消解体系，将有机络合态铬转化为可测的Cr(VI)离子（依据HJ 700-2014标准）。
2. **灰化**：测定农田土壤铅污染时，通过马弗炉550°C灼烧去除有机质，残留物用2%硝酸溶解后ICP-MS检测（避免铅挥发损失）。
3. **色谱**：使用DB-5毛细管柱分离水体中痕量有机氯农药，ECD检测器定量六六六异构体（γ-BHC检出限达0.01μg/L）。
