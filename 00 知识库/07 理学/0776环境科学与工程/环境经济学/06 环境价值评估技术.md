# 06 环境价值评估技术

## **6-1 为什么要评估？—— 为CBA提供弹药**

我们评估环境价值，不是真的要"卖掉"大自然，而是为了：

* **将环境影响纳入决策：** 只有把环境的价值"货币化"，才能放进CBA的框架里，与经济成本和效益进行比较，从而做出更明智的决策。
* **计算环境破坏的赔偿：** 比如石油泄漏事件，得算清楚对海洋生态系统造成了多少亿美元的损失，才能让肇事者赔偿。

## **6-2 价值的构成：总经济价值 (Total Economic Value, TEV)**

一个环境资源的总价值，可比我们想象的要复杂得多。经济学家把它分成了两大块：

* **总经济价值 ($TEV$) = 使用价值 ($UV$) + 非使用价值 ($NUV$)**

咱们用一个**国家森林公园**来举例子，你就全明白了：

1. **使用价值 (Use Value, UV):** 你能从**直接或间接使用**中获得的好处。
    * **直接使用价值 (Direct Use Value, DUV):** 你亲自去公园里能干的事。
        * *消耗性：* 砍伐木材、采摘药材、打猎。
        * *非消耗性：* 徒步、观鸟、露营、拍照。
    * **间接使用价值 (Indirect Use Value, IUV):** 公园"默默地"为你提供的好处，你甚至都没去过那里。
        * *生态功能：* 涵养水源、调节气候、净化空气、防止水土流失。
    * **选择价值 (Option Value, OV):** 现在不用，但保留未来可能使用的选择权。
        * "我现在不去这个公园，但我希望它一直在那里，万一我哪天想去了呢？" 这个"选择权"本身就有价值。

2. **非使用价值 (Non-Use Value, NUV):** 就算你这辈子都不打算去用，但它存在本身就让你感到满足。
    * **遗产价值 (Bequest Value, BV):** 为子孙后代保留这片森林的价值。
        * "我希望我的孙子孙女也能看到这片美丽的森林。"
    * **存在价值 (Existence Value, EV):** 仅仅是知道这个地方或这个物种存在，你就会感到高兴。
        * "我很高兴知道，遥远的亚马逊雨林里生活着各种神奇的生物，即使我永远也见不到它们。"

看明白了吧？TEV就像一个大洋葱，一层一层剥开，每一层都有其独特的价值。

## **6-3 评估的两大流派："侦探派" vs "民调派"**

那么，这些五花八门的价值到底怎么算出来呢？经济学家们分成了两大门派：

1. **显示偏好法 (Revealed Preference): "侦探派"**
    * **核心思想：** "别听他怎么说，要看他怎么做！" 通过观察人们在**真实市场**中的行为，来**间接推断**环境的价值。
    * **特点：** 基于真实数据，比较客观，但只能评估和市场行为相关联的**使用价值**。

2. **陈述偏好法 (Stated Preference): "民调派"**
    * **核心思想：** "既然没有市场，那我就创造一个！" 通过**问卷调查**等方式，构建一个**假想市场**，直接询问人们对环境改善**愿意支付多少钱 (WTP)** 或对环境破坏**愿意接受多少补偿 (WTA)**。
    * **特点：** 方法灵活，是唯一能评估**非使用价值**（如存在价值）的方法，但容易受到各种主观偏见的影响。

## **6-4 各派武功秘籍详解**

### **"侦探派"的独门绝技 (显示偏好法)**

* **生产率变动法 (Production Function Approach):**
    * **一句话原理：** 把环境质量看作是生产过程的一个"投入要素"。
    * **例子：** 空气污染加重 → 蜜蜂授粉效率降低 → 果园产量下降。果园减产的价值，就是空气污染造成的一部分损失。你的笔记里提到的土壤侵蚀导致玉米减产也是这个道理。
* **人力资本法 (Human Capital Approach):**
    * **一句话原理：** 环境污染影响健康，健康的价值可以用医疗费用和误工损失来衡量。
    * **例子：** 雾霾导致更多人患上呼吸道疾病。评估价值 = 增加的医药费 + 因病假而损失的工资。
    * **争议：** 这种方法因其伦理问题（难道富人的命比穷人更值钱？）而备受争议。
* **享乐定价法 (Hedonic Pricing Method):**
    * **一句话原理：** 环境好坏的价值，已经"隐藏"在了相关商品（主要是房地产）的价格里。
    * **例子：** 在其他条件（面积、户型、交通等）完全相同的情况下，一个紧邻公园的房子比一个靠近垃圾场的房子贵50万。这50万，就可以看作是优美环境和清新空气带来的价值。
* **旅行费用法 (Travel Cost Method):**
    * **一句话原理：** 人们愿意花多少钱（包括路费、门票、时间成本）去一个自然景点，就反映了这个景点的娱乐价值至少值多少钱。
    * **例子：** 张三花了500元路费和门票，还搭上了一整天周末（时间也要折算成钱），去一个森林公园玩。那么这个公园为张三提供的娱乐价值就至少是这几百块钱。

### **"民调派"的看家本领 (陈述偏好法)**

* **条件价值评估法 (Contingent Valuation Method, CVM):**
    * **一句话原理：** "直接问！"——用科学设计的问卷，在假想市场中直接问出人们的支付意愿(WTP)。
    * **你的笔记里那个"吉林大学清湖工程"就是完美的CVM案例！** 它的步骤非常经典：
        1. **构建假想市场：** 详细描述项目（清理湖水、修步道、绿化），让被调查者完全明白是在为什么东西"付费"。
        2. **设计支付方式 (Payment Vehicle):** 告诉他们钱会怎么"收"上来，比如"在学费里增加一项50元的年度环境附加费"。
        3. **引导出支付意愿：** 用不同的提问技巧（如开放式提问"你最多愿意付多少？"、二分选择"你愿意付50元吗？"）来获得WTP的数值。
        4. **分析数据，排除偏差：** 这是最难的一步！因为人们在回答时可能会有：
            * **假设性偏差：** 反正不是真掏钱，吹牛不上税。
            * **策略性偏差：** 为了让项目通过，故意报高价；或者因为反感付费形式，故意报0元。
            * **信息偏差：** 对项目了解不充分，随便报个数。
        5. **得出总价值：** 将样本的平均WTP乘以总人口，得出项目的总经济价值。
