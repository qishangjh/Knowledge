# 第二章 水的物理化学处理方法
## 第一节 水中粗大颗粒物质的去除
- **水中粗大颗粒物质**：粒径在1mm以上，包括砂粒、小卵石、砾石、树枝、菜叶、碎布、垃圾等。
- **去除方法**：常用物理处理法，包括：
  - **筛滤截流**
  - **重力沉降**
  - **离心分离**
### 一、格栅、筛网与微滤机
#### （一）格栅（Bar Screen）
##### 1. 组成与作用
- **组成**：栅条 + 排泥管
- **材料**：金属栅条、带钩塑料栅条或金属筛网
- **作用**：
  - 截留较粗大的悬浮物或漂浮物（如树枝、塑料袋、碎布等）
  - 防止堵塞水泵、管渠及沉淀池排泥管
  - 减轻后续处理构筑物的负荷，保障系统正常运行
##### 2. 安装地点
- 倾斜或直立安装在：
  - 污水沟渠
  - 泵房集水井进口
  - 污水处理厂进水口
  - 沉砂池前的格栅间或渠道中
##### 3. 格栅类型

| 分类方式 | 类型 | 特点 |
|----------|------|------|
| 按栅条净间距 | 粗格栅（50~100mm）<br>中格栅（10~40mm）<br>细格栅（3~10mm） | 用于不同处理阶段，控制截留颗粒大小 |
| 按清渣方式 | 人工格栅<br>机械格栅 | 人工适合小规模，机械适合大中型水厂 |
| 其他常见类型 | 链式、伸缩臂式、循环齿耙式、曲面机械式、旋转式、钢丝绳牵引式、液压式等 | 各具结构特点，适应不同工况 |

##### 4. 污物截留与处理
- **截留效率**：取决于栅条间隙宽度
- **污物含水率**：75%~85%
- **处理方式**：
  - 填埋
  - 卫生堆弃
  - 堆肥发酵
  - 焚烧
  - 与有机污泥混合后消化或粉碎再处理

---

#### （二）筛网与微滤机

##### 1. 筛网
- **孔径范围**：
  - 工业废水预处理：<10 mm
  - 最终水处理：<0.1 mm
- **应用**：进一步去除细小悬浮物

##### 2. 微滤机（Microstrainer）
- **特点**：
  - 孔径更小
  - 可高效截留细小悬浮物质
- **常用于**：污水处理的深度处理阶段

---
### 二、沉砂池（Grit Chamber）

#### 1. 作用
- 从污水中去除比重较大的无机颗粒（如砂子、煤渣等）
- 避免这些颗粒影响后续处理设备的运行
- 通常设在**泵站、沉淀池前**

#### 2. 工作原理
- **重力沉降**为基础
- 控制水流速度，使比重大的无机颗粒下沉
- 有机悬浮颗粒随水流带走

#### 3. 沉砂池类型

| 类型 | 结构特点 | 优缺点 |
|------|-----------|--------|
| 平流式 | 构造简单，水流水平流动 | 稳定性好，处理效果佳，排砂方便 |
| 竖流式 | 圆形池，水流自下而上 | 砂粒沉降效果一般 |
| 涡流式 | 利用旋转水流分离砂粒 | 分离效率高，占地小 |
| 曝气沉砂池 | 加入曝气，防止有机物腐败 | 除砂同时防臭，提高沉降效率 |

---

### **总结**

| 设备 | 主要功能 | 适用对象 | 处理粒径范围 |
|------|-----------|-----------|----------------|
| 格栅 | 截留粗大悬浮物 | 垃圾、树枝、碎布等 | >1mm |
| 筛网 | 截留较小悬浮物 | 工业废水预处理 | <10mm |
| 微滤机 | 截留极细悬浮物 | 深度处理 | <0.1mm |
| 沉砂池 | 去除比重大颗粒 | 砂子、煤渣等 | 0.2mm以上 |

## 第二节 水中悬浮物质和胶体物质的去除
### 一、沉淀法（Sedimentation）
#### （一）概述
- **定义**：利用水中颗粒的可沉降性，在重力作用下实现固液分离的过程。 #沉淀法
- **应用**：
  1. 预处理：沉砂池（Grit Chamber）
  2. 初沉池（Primary Settling Tank）：生物处理前
  3. 二沉池（Secondary Settling Tank）：生物处理后
  4. 污泥处理：污泥浓缩池（Sludge Thickener）

#### （二）沉淀类型

| 类型 | 特点 | 适用场景 |
|------|------|----------|
| **自由沉淀**<br>Discrete Particle Settling | 颗粒浓度低，互不干扰，独立沉降 | 沉砂池、初沉池 |
| **絮凝沉淀**<br>Flocculent Settling | 颗粒浓度中等（50~500 mg/L），相互碰撞形成絮体，沉速加快 | 二沉池中上部 |
| **区域沉淀**<br>Zone Settling | 颗粒浓度高（>500 mg/L），整体下沉，形成清晰界面 | 二沉池中下部 |
| **压缩沉淀**<br>Compression Settling | 颗粒堆积后，上层挤压下层水，污泥浓缩 | 二沉池泥斗、浓缩池 |

#### （三）理想沉淀池与沉淀效率分析
##### 1. 理想沉淀池结构
- **分区**：进口区 + 沉淀区 + 出口区 + 缓冲区 + 污泥区
- **基本假设**：
  - 水平流速均匀（v）
  - 颗粒下沉速度恒定（u）
  - 颗粒进入时均匀分布
  - 一旦沉到底即被去除

##### 2. 沉淀效率公式
- **临界沉速**：u₀ = vH / L
- **去除率**：η = ∫ (ut / u₀) f(ut) dut
- **表面负荷率**：q = Q / A = u₀

> **结论**：沉淀效率取决于颗粒沉速 ut 与表面负荷 q 的比值，与沉淀时间无关。

#### （四）沉淀池分类
#沉淀池分类

| 类型 | 结构特点 | 优点 | 缺点 |
|------|-----------|------|------|
| **平流式** | 长方形，水流缓慢 | 结构简单，处理稳定 | 占地大 |
| **竖流式** | 圆形，水流自下而上 | 适用于小厂 | 去除率低，池深大 |
| **辐流式** | 圆形，水流从中心或周边进出 | 处理量大，占地小 | 排泥设备复杂 |
| **斜板/斜管沉淀池** | 增设斜板/斜管，缩短沉降距离 | 效率高、占地小 | 易堵塞、维护难 |

#### （五）沉淀池选型考虑因素
- 水量大小
- 悬浮物浓度与沉降性能
- 地形、地质条件
- 施工与运行管理难度

#### （六）提高沉淀效率的方法
1. **曝气搅动**：促进自然絮凝，提高5~8%效率
2. **生物絮凝**：投加活性污泥，提高10~15%效率，BOD去除率提高15%以上。 **将剩余活性污泥回用**

### 二、混凝法（Coagulation and Flocculation）
#### （一）混凝原理
#混凝
##### 1. 定义
- **凝聚**（Coagulation）：使胶体脱稳并聚集为微絮粒
- **絮凝**（Flocculation）：微絮粒通过吸附、桥连形成大絮体
##### 2. 胶体稳定性
- **动力学稳定**：布朗运动抵抗沉降
- **聚集稳定**：同性电荷产生斥力，阻止聚集
##### 3. 胶团结构
- **胶核**：不溶性微粒
- **电位离子**：决定胶粒电荷
- **反离子层**：分为吸附层与扩散层
- **ξ电位**：滑动面上的电位，决定稳定性
##### 4. 颗粒间作用力
- **静电斥力**
- **范德华引力**
- **合力曲线**：存在"势垒"，阻碍聚集

#### （二）混凝机理

| 机理 | 描述 | 作用 |
|------|------|------|
| **压缩双电层** | 加入电解质降低ξ电位 | 胶体脱稳 |
| **吸附-电性中和** | 异号离子吸附降低电位 | 破坏稳定性 |
| **吸附架桥** | 高分子链状结构连接颗粒 | 形成大絮体 |
| **网捕作用** | 沉淀物包裹胶体颗粒 | 共同沉降 |

> 实际中多种机理共同作用，以某一为主。

#### （三）混凝剂与助凝剂
##### 1. 常用混凝剂（不考）
- **铝系**：硫酸铝、明矾、PAC、PAS
- **铁系**：三氯化铁、硫酸铁、PFS、PFC
- **有机高分子**：
  - 合成：PAM、HPAM、PEO
  - 天然：淀粉、树胶、甲壳素
- **发展方向**：复合型（铁铝复合、无机-有机复合）

##### 2. 助凝剂
- **酸碱类**：调节pH，如石灰、硫酸
- **结构改良类**：如活化硅酸、骨胶、PAM
- **氧化剂类**：破坏干扰物质，如Cl₂、O₃

#### （四）影响混凝效果的因素

| 因素        | 影响                                                     | 应对措施                                                            |
| --------- | ------------------------------------------------------ | --------------------------------------------------------------- |
| **水温**    | 低温影响水解、布朗运动、水化作用                                       | 增加投加量、投加高分子助凝剂                                                  |
| **pH值**   | 影响水解形态与混凝机理                                            | 控制在最佳pH范围                                                       |
| **碱度**    | 缓冲pH变化，维持水解反应                                          | 补充碱度                                                            |
| **悬浮物浓度** | 浓度低则碰撞少，混凝效果差；<br>浓度高则（采用高分子助凝剂）                       | 投加助凝剂、矿物颗粒、采用直接过滤（省去沉淀步骤）<br>聚合氯化铝对高浊水效果较好                      |
| **水力条件**  | **混合阶段**：快速均匀扩散，几秒至1分钟完成<br>**反应阶段**：缓慢搅拌，促进絮凝体成长，避免破碎 | 控制搅拌强度、时间<br>**异向絮凝**：由布朗运动造成的颗粒碰撞聚集<br>**同向絮凝**：由流体运动造成的颗粒碰撞聚集 |

### 三、澄清法（Clarification）
#### （一）基本原理
- **作用**：将**混凝处理的三个阶段**（混合、反应、分离）在**澄清池中一体化完成**。
- **关键机制**：**悬浮泥渣**吸附水中絮凝体颗粒，实现**接触絮凝**作用。
- **泥渣作用**：作为吸附和絮凝的核心，其**浓度稳定、分布均匀**是澄清效果的关键。
#### （二）澄清池类型

| 类型 | 工作原理 | 代表形式 |
|------|-----------|-----------|
| **泥渣循环分离型** | 泥渣在池中垂直循环，捕捉絮凝体并分离 | 机械加速澄清池<br>水力循环澄清池 |
| **悬浮泥渣过滤型** | 利用上升水流形成悬浮泥渣层，拦截絮凝体 | 普通悬浮澄清池<br>脉冲澄清池 |

### 四、过滤法（Filtration）
#### （一）定义
#粒状介质过滤定义
- 待处理水通过**粒状滤料**（如石英砂）床层时，其中的**悬浮颗粒**和被混**凝了的胶体物质**被截留在滤料的表面和内部空隙中，这种通过**粒状介质层**分离**不溶性污染物**的方法称为**粒状介质过滤**。
#### （二）应用
1. 化学混凝、生化处理后的**深度处理**
2. 活性炭吸附、离子交换等工艺的**前处理**

#### （三）过滤机理（三者共同作用）
#过滤机理

| 机理 | 描述 |
|------|------|
| **阻力截留** | 颗粒被滤料孔隙物理阻挡 |
| **重力沉降** | 滤料提供巨大沉降面积（如1m³滤料可有400m²沉降面积） |
| **接触絮凝** | 滤料表面电荷吸附胶体，形成多层吸附，增强去除效果 |

> **总结**：三种机理同时作用，提高去除效率。

### 五、气浮法（Flotation）
#### （一）基本原理
#气浮法
- **过程**：将**微小气泡**通入水中，与悬浮颗粒粘附，形成**密度小于水的气-颗粒复合体**，上浮至水面形成浮渣，从而实现分离。
- **适用对象**：轻质颗粒、油脂、浮渣、活性污泥等。

#### （二）气浮法的基本条件
1. 提供**足够量的微小气泡**
2. 悬浮物处于**悬浮状态**
3. 气泡与颗粒能**有效粘附**

#### （三）应用领域
1. 石油、化工、机械制造业的**油水分离**
2. 造纸废水中的**纸浆纤维回收**
3. 替代**二沉池**（尤其适用于污泥膨胀情况）
4. **剩余污泥浓缩**

#### （四）气浮法优点（与沉淀法比较）

| 优点 | 描述 |
|------|------|
| 占地小 | 表面负荷高，停留时间短，池深仅2m左右 |
| 出水含氧 | 具有预曝气作用，出水水质好，不易腐化 |
| 浮渣含水率低 | 一般在96%以下，体积小，便于后续处理 |
| 回收有用物质 | 如油脂、纤维等 |
| 药剂用量少 | 比沉淀法节省药剂 |

#### （五）缺点
- 电耗大（每吨水多耗电约0.02~0.04 kWh）
- 设备易堵塞（如溶气释放器）
- 浮渣易受风雨影响
### （六）化学药剂对气浮效果的影响

| 类型 | 功能 | 常用药剂 |
|------|------|----------|
| **混凝剂** | 促进颗粒絮凝，增强气泡粘附能力 | 铝盐、铁盐、PAC、PAM |
| **浮选剂** | 改变颗粒表面性质，增强疏水性 | 松香油、煤油、脂肪酸盐等 |
| **助凝剂** | 提高颗粒可浮性 | 聚丙烯酰胺（PAM） |
| **抑制剂** | 抑制某些物质上浮 | 石灰、硫化钠等 |
| **调节剂** | 调节pH值，改善气泡分散性 | 酸、碱等 |

### （七）气浮法类型（按气泡生成方式）

| 类型 | 原理 | 特点 |
|------|------|------|
| **电解气浮法** | 通电电解产生H₂和O₂气泡 | 气泡细小，适合脆弱絮体<br>电耗高，操作复杂 |
| **分散空气气浮法** | 机械搅拌或剪切产生气泡 | 常用于含油脂、表面活性剂废水 |
| **溶解空气气浮法** | 加压溶气后释放形成微气泡 | 气泡细小（20~100μm）<br>处理效果好，应用最广 |

#### 溶解空气气浮法流程
1. **全溶气流程**：全部原水加压溶气
2. **部分溶气流程**：部分原水加压溶气
3. **回流加压溶气流程**：回流部分处理水进行溶气

| 方法 | 原理 | 适用对象 | 优点 | 缺点 |
|------|------|----------|------|------|
| **澄清法** | 混凝+泥渣吸附 | 混凝处理全过程 | 一体化操作，效率高 | 对泥渣控制要求高 |
| **过滤法** | 滤料截留颗粒 | 深度处理 | 去除细小颗粒，出水清澈 | 易堵塞，需反冲洗 |
| **气浮法** | 气泡粘附颗粒上浮 | 轻质颗粒、油脂、浮渣 | 出水好，占地小，浮渣含水率低 | 电耗高，设备复杂 |

## 第三节 水中溶解物质的去除

### 一、溶解物质的种类
- **离子类**：Ca²⁺、Mg²⁺、K⁺、Na⁺、HCO₃⁻、SO₄²⁻、Cl⁻  
- **气体类**：O₂、CO₂、N₂  
- **复杂成分**：重金属离子、有机物（来自污水和废水）
### 二、主要去除方法概述
1. **软化除盐**：降低Ca²⁺、Mg²⁺含量，去除部分或全部盐类
2. **离子交换法**：选择性吸附水中离子
3. **吸附法**：利用多孔材料吸附溶解性杂质
4. **膜分离技术**：通过膜实现选择性透过
5. **中和法**：调节pH值处理酸碱废水
6. **氧化还原法**：转化有毒物质为无毒或低毒形式
7. **化学沉淀法**：通过沉淀去除重金属等

## 一、水的软化与除盐
### （一）软化方法
1. **加热软化法**：用于碳酸盐硬度
2. **药剂软化法**
   - 石灰法
   - 石灰-纯碱法
   - 石灰-石膏法

### （二）除盐方法
- 蒸馏法
- 电渗析法
- 离子交换法（最广泛）

## 二、离子交换法
### （一）概述
- 利用树脂上的离子与水中离子进行交换
- **应用**：软化、除盐、回收重金属、处理放射性废水等
- **优点**：去除率高、可回收物质、设备简单
- **缺点**：预处理要求高、树脂再生困难

### （二）离子交换树脂分类
| 类型       | 活性基团   | 举例    |
| -------- | ------ | ----- |
| 强酸性阳离子树脂 | -SO₃H  | 732树脂 |
| 弱酸性阳离子树脂 | -COOH  | 724树脂 |
| 强碱性阴离子树脂 | =NOH   | 717树脂 |
| 弱碱性阴离子树脂 | -NH₃OH | 710树脂 |
| 螗合树脂     | 胺羧基团   | 用于重金属 |
| 氧化还原树脂   | 氧化还原基团 | 特殊用途  |

### （三）树脂性能
- **物理性能**：粒径（0.3–1.2 mm）、密度、含水量、溶胀性、机械强度、耐热性
- **化学性能**：
  - 交联度（8–12%）
  - 酸碱性（强/弱酸碱）
  - 选择性（高价离子优先）
  - 交换容量（总容量 vs 工作容量）
### （四）操作流程
#离子交换操作过程
1. **交换（运行/吸附）**：
    - **目的**：去除水中目标离子
    - **过程**：待处理水流过树脂层，树脂捕获目标离子，释放替代离子。
    - **终点**：出水水质不达标时，树脂"饱和"，进入下一阶段。
2. **反洗（Backwash）**：
    - **目的**：**松动树脂，冲走杂质**。
    - **过程**：清水从底部向上冲洗树脂，使其膨胀，杂质排出。
    - **细节**：树脂膨胀40%~60%，持续约15分钟。
3. **再生（Regeneration）**：
    - **目的**：用高浓度再生液恢复交换能力
    - **过程**：高浓度再生液（如酸、碱、盐）流过树脂，把被吸附的离子"赶走"。
    - **细节**：再生剂种类、浓度、用量需精确控制。
4. **清洗（Rinse）**：
    - **目的**：**冲洗掉**树脂层中残留的**再生废液和杂质**，准备重新运行。
    - **过程**：用净水冲洗树脂，直到出水水质达标。
    - **细节**：用水量通常是树脂体积的4~13倍。
### （五）应用实例
- **电镀含铬废水**：732阳树脂 + 710阴树脂
- **含汞废水处理**：硼树脂 + 螯合树脂
- **其他应用**：回收贵重金属、放射性废水处理

## 三、膜分离技术

### （一）基本原理
- 利用**半透膜**选择性透过，实现溶质或溶剂的分离

### （二）膜分类
| 类型 | 分离机理 | 推动力 | 用途 |
|------|----------|--------|------|
| 渗析膜 | 浓度差 | 浓度差 | 分离小分子 |
| 电渗析膜 | 电位差 | 电位差 | 脱盐、回收酸碱 |
| 反渗透膜 | 压力差 | 压力差 | 海水淡化、去离子 |
| 超滤膜 | 压力差 | 压力差 | 去除大分子、微生物 |

### （三）膜材料
- **有机材料**：聚砜、聚酰胺、纤维素、硅橡胶
- **无机材料**：陶瓷、玻璃、金属（铝、钯、银）

### （四）膜分离特点
- **优点**：
  - 无相变、能耗低
  - 常温操作、适用于热敏物质
  - 分离效率高、设备简单
- **缺点**：
  - 易污染
  - 投资成本高
  - 对预处理要求高

## 四、吸附法
### （一）吸附原理
- 吸附剂表面存在**剩余力场**，可吸附液相中溶质
- **类型**：
  - 物理吸附（可逆、无选择性）
  - 化学吸附（不可逆、有选择性）
  - 离子交换吸附（静电作用）

### （二）常用吸附剂
1. **活性炭**
   - 粉末状：吸附快、再生难
   - 颗粒状：可再生、操作方便
2. **腐植酸类吸附剂**：用于重金属吸附（如汞、铬、铅等）

### （三）影响因素
- 吸附剂性质（比表面积、孔径、极性）
- 吸附质性质（极性、分子量、溶解度）
- 水质条件（pH、温度、共存物质）

### （四）吸附操作方式
- **静态吸附**：搅拌混合
- **动态吸附**：水流通过吸附剂层（固定床、移动床、流化床）

### （五）应用
- 去除有机物（COD、色度、异味）
- 去除重金属（铅、汞、铬等）

## 第四节 水的其他物理和化学处理方法
### 一、中和法
通过**加入酸性物质或碱性物质**，来**调节废水的pH值**，使其达到一个**中性范围（通常是pH 6～9）** 的过程。
- **酸 + 碱 → 盐 + 水**
#### （一）适用对象
- 酸性废水（电镀、化工等）
- 碱性废水（印染、造纸等）
#### （二）中和方法
1. **互相中和法**：酸碱废水混合
    - **① 用碱中和酸性废水**：向酸性废水中加入碱性试剂。
        - **常用碱性试剂**：
            - **氢氧化钠 (NaOH)**：俗称烧碱、火碱。效果快，易溶，但成本较高，腐蚀性强。
            - **石灰 (CaO 或 Ca(OH)₂)**：生石灰或熟石灰。
            - **碳酸钠 (Na₂CO₃)**：俗称纯碱。
            - **白云石、菱镁矿等**：天然矿物，成本低，但反应速度慢，需要磨碎。
    - **② 用酸中和碱性废水**：
        - **原理**：向碱性废水中加入酸性试剂。
        - **常用酸性试剂**：
            - **硫酸 (H₂SO₄)**：工业上最常用的酸，价格便宜，但腐蚀性强。
            - **盐酸 (HCl)**：反应速度快，不易结垢。但成本相对较高，有挥发性，对设备腐蚀性也强。
            - **二氧化碳 (CO₂) 或烟道气**：
                - 可以利用工业锅炉排出的烟道气（含有约10%～20%的CO₂、SO₂）
                - 反应原理：$CO_2 + H_2O \rightleftharpoons H_2CO_3$； $H_2CO_3 + 2NaOH \rightarrow Na_2CO_3 + 2H_2O$
2. **药剂中和法**：
   - 酸性废水：石灰、石灰石
   - 碱性废水：硫酸、盐酸
3. **过滤中和法**：使用石灰石、白云石滤料
4. 碱性废水的中和处理：
	1. 药剂中和
	2. 烟道气中和

| 废水性质 | 常用中和剂 | 优点 | 缺点 | 注意事项 |
| :------- | :--------- | :--- | :--- | :------- |
| **酸性废水** | 氢氧化钠 (NaOH) | 反应快，易溶 | 成本高，腐蚀性强 | 剧烈放热，需冷却 |
| | 石灰 (CaO/Ca(OH)₂) | 价格便宜，助沉淀 | 沉淀量大，易结垢 | 易堵塞管路 |
| | 碳酸钠 (Na₂CO₃) | 反应温和 | 成本略高 | 溶解度适中 |
| **碱性废水** | 硫酸 (H₂SO₄) | 价格低，易获取 | 强腐蚀，易结垢 (CaSO₄) | 避免与高钙废水混合 |
| | 盐酸 (HCl) | 反应快，不易结垢 | 成本高，有挥发性 | 需注意安全防护 |
| | 二氧化碳 (CO₂) 或烟道气 | 经济环保 | 反应速度较慢，适用于弱碱性 | 需要曝气设备 |

### 二、氧化还原法
#### （一）原理
- 将有毒物质通过氧化或还原转化为无毒或低毒物质

#### （二）常用药剂
- **氧化剂**：O₃、H₂O₂、Cl₂、次氯酸钠
- **还原剂**：FeSO₄、Na₂SO₃、Zn粉、SO₂

#### （三）应用
- **臭氧氧化**：脱色、杀菌
- **过氧化氢氧化**：处理有机污染物
- **光化学氧化**：UV+H₂O₂
- **湿式氧化**：高温高压处理难降解有机物
- **电解法**：电化学氧化还原、气浮、凝聚

### 三、化学沉淀法

#### （一）原理
- 利用溶度积原理，使金属形成难溶沉淀物
#### （二）常见方法
1. **氢氧化物沉淀法**：适用于Zn、Pb、Cr、Al（注意两性）
2. **硫化物沉淀法**：比氢氧化物更彻底，但易造成二次污染
3. **碳酸盐沉淀法**：用于回收Zn、Cu、Pb
4. **化学沉淀除磷**：如石灰、铁盐、铝盐

## 第五节 水中有害微生物的去除
### 一. 水中的细菌和病毒
各类水体是繁衍微生物生长的天然环境。大多数无害，少数病原微生物很有害。  流通途径：垃圾、粪便、污水等水至传染病或水性传染病  肠道伤寒霍乱痢疾病毒性肝炎螺旋体病脊髓灰质炎等。寄生虫病
### 二.水的消毒
给水处理中，混凝沉淀和过滤都能消除不少细菌和微生物，但不能去除所有病原微生物。 1.消毒的目的：杀死部分！ 与灭菌不同,并不是杀死一切微生物，而是只杀有害病原微生物。 2.方法：煮沸法氯消毒臭氧消毒二氧化氯消毒紫外线消毒
