二、实验目的

1. 掌握COD测定的步骤及注意事项，能够独立准确地进行滴定操作。
2. 理解COD的测定原理，能够依据实验数据精确计算COD浓度。
3. 学会正确使用实验仪器，规范完成实验流程。

三、实验原理  
在水样分析中，加入已知量的重铬酸钾（K₂Cr₂O₇）溶液，在强酸性介质（H₂SO₄）和银盐（Ag₂SO₄）催化下进行回流反应。芳烃（如C₆H₆）及吡啶（C₅H₅N）在此条件下氧化率较低，难以完全被氧化；而直链脂肪族化合物（如CH₃(CH₂)ₙCOOH）在Ag₂SO₄催化作用下可有效氧化。

反应结束后，以试亚铁灵为指示剂，用硫酸亚铁铵[(NH₄)₂Fe(SO₄)₂]滴定未被还原的Cr₂O₇²⁻。根据滴定消耗的硫酸亚铁铵量，计算出水样中还原性物质消耗氧的质量浓度（COD）。其基本反应如下：

1. 氧化反应：  
$\text{有机物} + \text{Cr}_2\text{O}_7^{2-} + 14\text{H}^+ + 6e^- \rightarrow 2\text{Cr}^{3+} + 7\text{H}_2\text{O} + \text{CO}_2$

2. 滴定反应：  
$\text{Cr}_2\text{O}_7^{2-} + 6\text{Fe}^{2+} + 14\text{H}^+ \rightarrow 2\text{Cr}^{3+} + 6\text{Fe}^{3+} + 7\text{H}_2\text{O}$

五、实验步骤

（一）稀释  
由于水样COD值较高，需对水样进行稀释。取5.00mL水样，稀释至100mL定容。

（二）消解  
取水样20mL于250mL锥形瓶中，加入0.4g硫酸汞固体，加入20.00mL重铬酸钾溶液，同时加入3粒玻璃珠。加入玻璃珠时，应倾斜锥形瓶，使玻璃珠缓慢滚入，防止打破锥形瓶。稍摇匀后，连接至COD消解仪。从已经注入冷凝水的冷凝管上口加入20mL硫酸-硫酸银溶液，一定要缓慢加入，并尽可能轻轻摇动锥形瓶。同时设置空白实验，面板设置300℃，120min，开始消解。

（三）滴定  
待消解样品冷却后，从上口加入30mL水，待再次冷却后，将锥形瓶取下，加入40mL水摇匀待用。向锥形瓶加入3滴指示剂，用硫酸亚铁铵进行滴定。空白样品步骤同上。

六、课前实验设计  
在课前，根据实验指导书要求，准备相应的实验试剂和溶液配制。具体如下：

- 配制重铬酸钾标准溶液c（1/6K₂Cr₂O₇）=0.0250mol/L：将7.3545g在105℃干燥2h后的重铬酸钾溶于水中，稀释至1000mL。
- 配制硫酸亚铁铵溶液c（（NH₄）₂Fe（SO₄）₂·6H₂O）=0.25mol/L：称取98g硫酸亚铁铵于水中，加入20mL浓硫酸，稀释定容至1000mL。由于课时有限，所有组别不再进行硫酸亚铁铵标定操作，统一用课前预实验标定浓度0.2419mol/L进行计算。
- 配制硫酸-硫酸银溶液：向500mL浓硫酸中加入5g硫酸银，搅拌溶解后备用。
- 配制试亚铁灵指示剂：0.7g七水合硫酸亚铁（FeSO₄·7H₂O）于50mL水中，加入1.5g邻菲罗啉，搅拌至溶解，定容至100mL。

八、数据处理  
化学需氧量以mg/L计，计算如下：  
COD（mg/L）=C(V₁−V₂)×8000/V₀  
其中：

- C——硫酸亚铁铵标准溶液浓度，mol/L；
- V₁——空白试验所消耗的硫酸亚铁铵标准溶液的体积，mL；
- V₂——试样滴定所消耗的硫酸亚铁铵标准溶液的体积，mL；
- V₀——样品体积，mL。  
计算结果要相应进行稀释浓度的换算。

| 样品  | V₁（mL） | V₂（mL） | C（mol/L） | V₀（mL） | COD（mg/L） |
| --- | ------ | ------ | -------- | ------ | --------- |
| 空白  |        |        | 0.2419   | 20.00  |           |
| 水样  |        |        |          | 20.00  |           |

九、结果讨论  
在本次实验中，我们通过滴定法测定了水样中的COD值。在实验过程中，我们严格按照实验步骤进行操作，确保了实验的准确性和可靠性。然而，在实验中也出现了一些问题，例如在滴定过程中，指示剂的颜色变化不够明显，可能会影响滴定终点的判断。此外，实验中的一些操作步骤，如消解温度和时间的控制，也可能对实验结果产生影响。  
如果再次进行实验，我会更加注意以下几点：

1. 严格控制消解温度和时间，确保水样中的有机物能够充分氧化。
2. 在滴定过程中，更加仔细地观察指示剂的颜色变化，确保滴定终点的准确判断。
3. 对实验数据进行多次测量和分析，减少实验误差，提高实验结果的准确性。
