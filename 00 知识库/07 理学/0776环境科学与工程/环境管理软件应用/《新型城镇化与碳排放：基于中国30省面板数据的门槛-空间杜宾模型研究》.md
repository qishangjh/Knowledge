# 《新型城镇化与碳排放：基于中国30省面板数据的门槛-空间杜宾模型研究》

---

## 一、文献综述（整合创新）

| 研究维度 | 传统城镇化 | 新型城镇化（陈涵康） | 门槛效应（王雅楠） | 空间效应（徐英启） |
|----------|-------------|----------------------|----------------------|----------------------|
| **人口** | 粗放扩张 → 碳↑ | 人口质量↑ → 碳↓ | 人均GDP门槛：倒U型 | 溢出效应微弱 |
| **经济** | 二三产扩张 → 碳↑ | 三产主导 → 碳↓ | 能源消费门槛：倒U型 | 无空间溢出 |
| **空间** | 无序蔓延 → 碳↑ | 集约规划 → 碳↓ | 未涉及 | 抑制本地碳排放 |
| **生态** | 忽视绿化 → 碳↑ | 碳汇建设 → 碳↓ | 未涉及 | 无空间溢出 |

**研究空白**：现有研究多聚焦线性关系或单一国家尺度，缺乏对典型区域城镇化与碳排放非线性关系的深入探讨，且对空间异质性的量化分析不足。

---

## 二、模型构建（整合方法）

### 2.1 研究区域与数据
- **区域**：中国30省（2012-2021年，不含西藏/港澳台）
- **数据**：
  - **碳排放**：IPCC表观核算法（煤炭、焦炭、天然气等7类能源）
  - **新型城镇化**：22指标 → 熵权法 → 五维度得分（人口/经济/空间/生态/社会）
  - **控制变量**：人口规模（P）、人均收入（A）、能源强度（T）

### 2.2 模型设计
**Step1：门槛模型**（验证非线性）  
\[  
\ln CO_{2,it} = \alpha_i + \beta_1 \ln UR_{it} \cdot I(q_{it} \leq \gamma) + \beta_2 \ln UR_{it} \cdot I(q_{it} > \gamma) + \delta \ln X_{it} + \mu_{it}  
\]
- **门槛变量**：人均GDP（q₁）、能源强度（q₂）
- **解释变量**：UR为五维度城镇化得分（分别回归）

**Step2：空间杜宾模型**（验证溢出）  
\[  
\ln CO_{2,it} = \rho W \ln CO_{2,it} + \beta \ln UR_{it} + \theta W \ln UR_{it} + \gamma \ln X_{it} + \epsilon_{it}  
\]
- **空间矩阵**：经济-地理嵌套矩阵（式6-1）
- **检验**：Moran's I → LR/Wald/Hausman → 时间固定效应SDM

以下是可直接运行的 **Stata 代码模板**（整合门槛回归 + 空间杜宾模型），用于分析 **新型城镇化五维度对碳排放的影响**。代码基于陈涵康（2024）与王雅楠（2018）的方法，适配中国30省2012-2021年面板数据。

---

### **1. 数据准备与变量定义**

```stata
* 设置工作目录
cd "D:\Research\Urbanization_CO2"

* 导入数据（需包含以下变量）
use "UR_CO2_panel2021.dta", clear

* 变量说明：
* province: 省份代码  year: 年份
* co2: 碳排放量（万吨）  p: 人口规模  a: 人均收入  t: 能源强度
* ur_pop: 人口城镇化得分  ur_econ: 经济城镇化得分
* ur_space: 空间城镇化得分  ur_eco: 生态城镇化得分  ur_soc: 社会城镇化得分

* 生成对数变量
foreach var in co2 p a t ur_pop ur_econ ur_space ur_eco ur_soc {
    gen ln`var' = ln(`var')
}

* 设置面板数据
xtset province year
```

---

### **2. 门槛效应检验（Hansen, 1999）**
#### **2.1 单门槛检验（以人口城镇化为例）**

```stata
xtthres ln_co2 ln_ur_pop ln_p ln_a ln_t, thres(ln_a) reps(300) seed(123)
* 结果解读：若F统计量显著，则存在门槛效应
```

#### **2.2 双门槛检验**

```stata
xtthres ln_co2 ln_ur_pop ln_p ln_a ln_t, thres(ln_a) reps(300) nthresholds(2)
```

#### **2.3 可视化门槛值**

```stata
xtthresgraph, xline(7.34) title("人均GDP门槛效应（人口城镇化）")
```

---

### **3. 空间杜宾模型（SDM）**
#### **3.1 空间权重矩阵**

```stata
* 生成经济-地理嵌套矩阵（需提前准备经纬度坐标）
spmat idistance W_geo longitude latitude, id(province) d(1) normalize
spmat summarize W_geo

* 经济距离矩阵（以人均GDP差异为权重）
spmat euclid W_econ, id(province) xcoord(ln_a) ycoord(ln_a) normalize
spmat summarize W_econ
```

#### **3.2 空间自相关检验（Moran's I）**

```stata
spatgsa ln_co2, weights(W_geo) moran
```

#### **3.3 模型选择检验**

```stata
* LM检验（SLM vs SEM）
xtmoran ln_co2 ln_p ln_a ln_t ln_ur_pop, model(lag)
xtmoran ln_co2 ln_p ln_a ln_t ln_ur_pop, model(error)

* LR检验（SDM vs SLM/SEM）
spxtregress ln_co2 ln_p ln_a ln_t ln_ur_pop, fe dvarlag(W_geo) ivarlag(W_geo: ln_p ln_a ln_t ln_ur_pop)
est store SDM
spxtregress ln_co2 ln_p ln_a ln_t ln_ur_pop, fe dvarlag(W_geo)
est store SLM
lrtest SDM SLM
```

#### **3.4 运行SDM（以人口城镇化为核心变量）**

```stata
spxtregress ln_co2 ln_p ln_a ln_t ln_ur_pop, fe ///
    dvarlag(W_geo) ivarlag(W_geo: ln_p ln_a ln_t ln_ur_pop) ///
    effects
```

#### **3.5 分解直接/间接/总效应**

```stata
estat impact, direct indirect total
```

### **5. 完整循环（五维度批量分析）**

```stata
foreach dim in pop econ space eco soc {
    spxtregress ln_co2 ln_p ln_a ln_t ln_ur_`dim', fe ///
        dvarlag(W_geo) ivarlag(W_geo: ln_p ln_a ln_t ln_ur_`dim') effects
    estat impact, direct indirect total
    est store SDM_`dim'
}
esttab SDM_pop SDM_econ SDM_space SDM_eco SDM_soc using "SDM_all.rtf", ///
    b(%7.3f) se starlevels(* 0.1 ** 0.05 *** 0.01)
```

### **6. 注意事项**
1. **数据要求**：需包含省份代码、年份、经纬度坐标。
2. **软件依赖**：安装 `xtthres`（门槛回归）、`spmat`（空间矩阵）、`spxtregress`（空间计量）。
3. **扩展**：替换 `ln_ur_pop` 为其他维度变量即可重复分析。
