# 单位换算
## 一、最基础的单位：位 (bit) 和 字节 (Byte)

这是所有计算机数据量的最基本构成单位。

1.  **位 (bit)**
    *   **英文**：`bit` (Binary digit 的缩写)
    *   **含义**：计算机中最小的数据单位，一个**二进制位**。它只能表示两种状态：`0` 或 `1`（开或关，有电或无电）。
    *   **通常表示**：用小写 `b` 表示，如 `Mb` (兆位)。
    *   **在408中**：数据传输速度常用位表示，如 `Mbps` (兆位每秒)。

2.  **字节 (Byte)**
    *   **英文**：`Byte`
    *   **含义**：计算机存储数据的基本单位，通常由 **8 个二进制位 (bit)** 组成。
    *   **通常表示**：用大写 `B` 表示，如 `MB` (兆字节)。
    *   **在408中**：存储容量、文件大小、内存地址等常用字节表示，如 `MB`、`GB`。

3.  **位与字节的换算**：
    *   **`1 Byte = 8 bit`**
    *   **`1 B = 8 b`**

    **【记忆口诀】**：**大B八个小b** (大写B是字节，小写b是位，一个字节等于八个位)。

---

## 二、存储容量单位 (基于字节的倍数)

这些单位是衡量存储设备（硬盘、内存、Cache）大小的。在计算机领域，这些单位通常是**2的幂次**，但有时也会被误用为10的幂次，需要注意区分。

| 单位名称 | 英文缩写 | 换算关系 (2的幂次) | 十进制近似 (10的幂次) |
| :------- | :------- | :------------------ | :---------------------- |
| **字节** | B        | 1 B                 | 1 B                     |
| **千字节** | KB       | 1 KB = 1024 B       | ≈ 10^3 B (1000 B)       |
| **兆字节** | MB       | 1 MB = 1024 KB      | ≈ 10^6 B (1,000,000 B)  |
| **吉字节** | GB       | 1 GB = 1024 MB      | ≈ 10^9 B                |
| **太字节** | TB       | 1 TB = 1024 GB      | ≈ 10^12 B               |
| **拍字节** | PB       | 1 PB = 1024 TB      | ≈ 10^15 B               |

*   **1024 是 `2^10`**。所以：
    *   `1 KB = 2^10 B`
    *   `1 MB = 2^10 KB = 2^20 B`
    *   `1 GB = 2^10 MB = 2^30 B`
    *   `1 TB = 2^10 GB = 2^40 B`

**【408考研注意】**：在考研题目中，通常默认使用 `1024` (即 `2^10`) 进行换算。除非题目明确说明 `1KB = 1000B`。

## 三、数据传输速率单位 (基于位或字节每秒)

这些单位是衡量数据传输速度的，通常与时间单位"秒"结合。

1.  **bps (bits per second)**
    *   **含义**：位每秒，表示每秒传输的二进制位数。
    *   **常见表示**：`Kbps` (千位每秒)、`Mbps` (兆位每秒)、`Gbps` (吉位每秒)。
    *   **用途**：常用于描述网络带宽、Modem 速度等。电信运营商通常用 `Mbps` 来标称网速。
    *   **换算**：这里的 `k`、`M`、`G` 通常是基于 **1000** (10的幂次) 进行换算，因为这是电信行业的习惯。
        *   `1 Kbps = 1000 bps`
        *   `1 Mbps = 1000 Kbps = 10^6 bps`
        *   `1 Gbps = 1000 Mbps = 10^9 bps`

2.  **B/s (Bytes per second)**
    *   **含义**：字节每秒，表示每秒传输的字节数。
    *   **常见表示**：`KB/s` (千字节每秒)、`MB/s` (兆字节每秒)、`GB/s` (吉字节每秒)。
    *   **用途**：常用于描述硬盘读写速度、文件下载速度等。操作系统显示文件传输速度时常用 `MB/s`。
    *   **换算**：这里的 `K`、`M`、`G` 同样通常是基于 **1000** (10的幂次) 进行换算，但有时也会使用 `1024`。在考研题目中，根据语境和问题目的，需要灵活判断。如果题目涉及到硬盘、总线带宽等，更倾向于使用 `1000` 或明确给出换算关系。
        *   `1 KB/s = 1000 B/s`
        *   `1 MB/s = 1000 KB/s = 10^6 B/s`

3.  **位/秒 与 字节/秒 的换算**：
    *   **`1 B/s = 8 bps`** (每秒传输一个字节，就相当于每秒传输八个位)
    *   **`1 MB/s = 8 Mbps`** (如果 M 是 10^6)
    *   **`1 MB/s = (1000 * 1000 * 8) bps`**
    *   **`1 Mbps = 1/8 MB/s`**

**【408考研再次注意】**：
*   如果题目提到**存储容量**（内存、硬盘大小），`K`、`M`、`G` 通常是 `1024`。
*   如果题目提到**网络带宽**（`Mbps`），`k`、`M`、`G` 通常是 `1000`。
*   如果题目提到**总线带宽、硬盘读写速度**（`MB/s`），换算 `K`、`M`、`G` 时，需要**结合实际语境判断，或留意题目是否有明确说明**。很多时候会默认 `1000`，但在精确计算时，为避免歧义，题目通常会给出 `1K=1024` 或 `1M=1000K`。如果实在没有，用 `1024` 会更"计算机化"，但也要做好两种情况的心理准备。**最稳妥是先按1024算，如果答案不对再考虑1000。**

## 四、其他常见单位

*   **Hz (Hertz)**：赫兹，表示频率。在计算机中常用于：
    *   **CPU 主频**：`GHz` (吉赫兹)，表示 CPU 每秒完成的时钟周期数。`1 GHz = 10^9 Hz`。
    *   **总线时钟频率**：`MHz` (兆赫兹)，`1 MHz = 10^6 Hz`。
    *   **`1 Hz = 1 次/秒`**。
*   **ns (nanosecond)**：纳秒，时间单位。`1 ns = 10^(-9) 秒`。常用于表示访存时间、指令执行时间。
*   **ms (millisecond)**：毫秒，时间单位。`1 ms = 10^(-3) 秒`。常用于表示硬盘寻道时间、I/O 操作时间。

**重点提示：**

1.  **FLOPS系列**：`MFLOPS` 到 `ZFLOPS` 是考研大纲明确要求的性能指标，用于衡量每秒浮点运算次数。它们都使用十进制前缀。
2.  **MIPS**：`MIPS` (Millions of Instructions Per Second) 衡量每秒百万条指令数，也使用十进制 `Mega` 前缀。
3.  **十进制与二进制前缀**：
    *   在表示**性能指标**（如MIPS, FLOPS, GHz等）、**频率**、**速率**时，前缀 K, M, G, T, P, E, Z, Y 均表示**十进制**的 $10^3, 10^6, 10^9, \dots$。
    *   在表示**存储容量**（如KB, MB, GB, TB）时，前缀 K, M, G, T 通常表示**二进制**的 $2^{10}, 2^{20}, 2^{30}, 2^{40}$ 等（即1024的幂次）。但在408中，如果题目未特殊说明，涉及到性能计算时，通常按十进制处理。在计算存储地址和容量时，则按照 $2^{10}$ 来计算。
4.  **中文传统单位**：作为辅助记忆和理解，可以了解“兆”对应 $10^6$， “吉”对应 $10^9$， “太”对应 $10^{12}$ 等，但考研题目中主要使用国际前缀符号。
