# 02 一维随机变量及其分布
---

## 02-1 一维随机变量：从事件到数值的飞跃

### 1. 概念
* **随机变量**：将样本空间中的每一个基本事件映射到实数集上的函数。
    * 你可以理解为，它把随机试验的结果"数值化"了，方便我们进行数学分析。
* **一维随机变量**：只取一个数值的随机变量，通常记为$X, Y, Z$等。
* 随机变量分为两类：
    * **离散型随机变量**：取值有限或可列。比如抛硬币正面的次数（0, 1），一天内接到的电话数量（0, 1, 2, …）。
    * **连续型随机变量**：取值在某个区间内连续。比如一个人的身高，公交车等待时间。

### 2. 分布函数（Cumulative Distribution Function, CDF）
#分布函数
#### (1) 概念
* 分布函数定义为：
    $$ F(x) = P(X \leq x) $$
    表示随机变量 $X$ 取值**小于等于**某个特定值 $x$ 的概率。它是一个描述随机变量整体分布情况的函数。

#### (2) 性质 (非常重要，常用于判断一个函数是否为分布函数)
1. **单调不减**：若$x_1 < x_2$，则$F(x_1) \leq F(x_2)$。说明概率是累积的，不会减少。
2. **有界性**：$0 \leq F(x) \leq 1$。因为概率的取值范围就是 $[0, 1]$。
3. **极限性质**：
    $$ \lim_{x \to -\infty} F(x) = 0, \quad \lim_{x \to +\infty} F(x) = 1 $$
    说明在负无穷处概率累积为0，在正无穷处概率累积为1。
4. **右连续性**：$\lim_{h \to 0^+} F(x + h) = F(x)$。这是分布函数的一个数学特性，意味着函数图像从右侧趋近某点时，函数值等于该点的函数值。

#### (3) 应用——求概率 (考研必考！)
* $P(a < X \leq b) = F(b) - F(a)$。这是分布函数最核心的应用之一。
* $P(X > a) = 1 - F(a)$。
* $P(X < a) = F(a^-)$（左极限）。
    * **注意**：对于连续型随机变量，$P(X < a) = P(X \leq a) = F(a)$，因为 $P(X=a) = 0$。
    * 对于离散型随机变量，$P(X < a)$ 通常不等于 $F(a)$，需要从 $F(a)$ 中减去 $P(X=a)$。

## 02-2 一维离散型随机变量：点状分布

### 1. 分布律（Probability Mass Function, PMF）
* 对于离散型随机变量$X$，其分布律为：
    $$ P(X = x_i) = p_i, \quad i = 1, 2, \dots $$
    其中$x_i$是$X$的可能取值，$p_i$是$X$取值$x_i$的概率。

### 2. 性质
1. 非负性：$p_i \geq 0$。
2. 归一性：$\sum_i p_i = 1$。所有可能取值的概率和为1。

### 3. 应用——求概率
* $P(a \leq X \leq b) = \sum_{x_i \in [a,b]} p_i$。直接将指定区间内所有取值的概率相加即可。

### 4. 五大常见离散型分布

#### (1) 0-1分布（伯努利分布）
* **概念**：只有两个结果：成功（1）或失败（0）的单次试验。
* **参数**：$p \in (0,1)$ (成功概率)。
* **分布律**：$P(X=1) = p, \quad P(X=0) = 1 - p$。
* **数字特征**：
    * **$E(X) = p$**
    * **$D(X) = p(1-p)$**

#### (2) 二项分布 $X \sim B(n, p)$
* **概念**：重复独立伯努利试验$n$次，成功$k$次的概率。
* **参数**：$n$ (试验次数), $p \in (0,1)$ (单次成功概率)。
* **分布律**：$P(X = k) = C_n^k p^k (1-p)^{n-k}, \quad k = 0,1,\dots,n$。
* **数字特征**：
    * **$E(X) = np$**
    * **$D(X) = np(1-p)$**

#### (3) 泊松分布 $X \sim P(\lambda)$
* **概念**：描述单位时间内（或单位面积内）发生稀有事件的次数。
* **参数**：$\lambda > 0$ (单位时间/面积内事件发生的平均次数)。
* **分布律**：$P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!}, \quad k = 0,1,2,\dots$。
* **数字特征**：
    * **$E(X) = \lambda$**
    * **$D(X) = \lambda$** (期望和方差相等是泊松分布的独特之处！)

#### (4) 几何分布 $X \sim G(p)$
* **概念**：在重复独立伯努利试验中，首次成功出现在第$k$次试验的概率。
* **参数**：$p \in (0,1)$ (单次成功概率)。
* **分布律**：$P(X = k) = (1-p)^{k-1}p, \quad k = 1,2,\dots$。
* **数字特征**：
    * **$E(X) = \frac{1}{p}$**
    * **$D(X) = \frac{1-p}{p^2}$**

#### (5) 超几何分布 $X \sim H(N, M, n)$
* **概念**：从总数为$N$的有限总体中**不放回**抽取$n$个，其中有$M$个是"成功"物品，抽到$k$个成功物品的概率。
* **参数**：$N$ (总体大小), $M$ (总体中"成功"的个数), $n$ (样本大小)。
* **分布律**：$P(X = k) = \frac{C_M^k C_{N-M}^{n-k}}{C_N^n}$。
    * 其中 $\max\{0, n-(N-M)\} \le k \le \min\{n, M\}$。
* **数字特征**：
    * **$E(X) = n \frac{M}{N}$**
    * **$D(X) = n \frac{M}{N} \frac{N-M}{N} \frac{N-n}{N-1}$** (比二项分布多了一个修正因子 $\frac{N-n}{N-1}$)。

## 02-3 一维连续型随机变量
### 1. 概率密度函数（PDF）
* 定义：若存在非负函数$f(x)$，使得 $F(x) = \int_{-\infty}^x f(t) dt$，则称$f(x)$是$X$的**概率密度函数**。
    * 概率密度函数本身不是概率，它描述了概率在实数轴上的**分布"密度"**。
    * $F'(x) = f(x)$ (在 $f(x)$ 连续的区间内)。

### 2. 性质 (非常重要，常用于判断一个函数是否为概率密度函数)
1. 非负性：$f(x) \geq 0$。
2. 归一性：$\int_{-\infty}^{\infty} f(x) dx = 1$。整个函数曲线下的面积为1。
3. 概率计算：$P(a < X \leq b) = \int_a^b f(x) dx$。区间概率通过积分计算。

### 3. 应用——求概率
* 利用积分计算区间概率。
* 某点概率为 0：$P(X = a) = 0$。这是连续型随机变量的一个重要特性，意味着随机变量取到某个精确值的概率为零。

### 4. 三大常见分布
#### **(1) 均匀分布 $X \sim U(a, b)$**

*   **概念**：随机变量 $X$ 在给定区间 $[a, b]$ 上取任意值都是**等可能的**。在该区间内概率密度函数值恒定。
    *   **注意**：这里的区间 $[a,b]$ 可以是开区间、半开半闭区间（如 $(a,b)$, $[a,b)$, $(a,b]$），对连续型随机变量而言，**不影响其概率计算**，因为单点处的概率为0。
*   **参数**：$a$ (区间下限), $b$ (区间上限)，且 $a < b$。
*   **概率密度函数 (PDF)**：
    $$ f(x) = \begin{cases}
    \frac{1}{b-a}, & a \leq x \leq b \\
    0, & \text{其他}
    \end{cases} $$
    * 也可以用示性函数表示为 $f(x) = \frac{1}{b-a} I(a \le x \le b)$。
*   **数字特征**：
    *   **期望 $E(X) = \frac{a+b}{2}$** （就是区间的中心点，很直观吧！）
    *   **方差 $D(X) = \frac{(b-a)^2}{12}$** （记住这个"12"，考研高频考点，直接用！）
*   **核心性质**：
    *   **区间外概率为零**：落在区间 $[a,b]$ 外部的概率为零，即 $P\{X < a\} = 0$ 和 $P\{X > b\} = 0$。
    *   **子区间概率与长度成正比**：落在 $[a,b]$ 内的任意子区间 $[c, d]$ 的概率，只与子区间的长度 $(d-c)$ 有关，与子区间的位置无关，且正比于长度：
        $$ P\{c \leq X \leq d\} = \int_c^d \frac{1}{b-a} \, dx = \frac{d-c}{b-a} $$
        *   这条性质是均匀分布"公平公正"的体现！

#### **(2) 指数分布 $X \sim Exp(\lambda)$ (考研重点！无记忆性是考点中的考点！)**
*   **概念**：通常用于描述**事件发生的时间间隔**，例如无线电元件的寿命、动物的寿命、随机服务系统服务时间等。它具有**无记忆性**。
    *   想象一个灯泡的寿命。无记忆性就是说，无论这个灯泡已经亮了多久，它在未来某段时间内发生故障的概率，都和它刚开始亮时一样。**"它不记得过去，只关注当下和未来"**。
*   **参数**：$\lambda > 0$ (速率参数，表示单位时间内事件平均发生的次数)。
*   **概率密度函数 (PDF)**：
    $$ f(x) = \begin{cases}
    \lambda e^{-\lambda x}, & x > 0 \\
    0, & x \le 0
    \end{cases} $$
*   **分布函数 (CDF)**：
    $$ F(x) = \begin{cases}
    1 - e^{-\lambda x}, & x > 0 \\
    0, & x \le 0
    \end{cases} $$
    * CDF 公式在求概率时超级方便 $P(X \le x) = F(x)$，$P(X > x) = 1 - F(x) = e^{-\lambda x}$。
*   **数字特征**：
    *   **期望 $E(X) = \frac{1}{\lambda}$**
    *   **方差 $D(X) = \frac{1}{\lambda^2}$**
*   **核心性质**：
    *   **无记忆性 (Memoryless Property)**：对于任意 $s, t > 0$，有
        $$ P(X > s+t \,|\, X > s) = P(X > t) $$
        *   这就是"灯泡不记得过去"的数学表达。在已知事件已经发生 $s$ 时间的前提下，它在未来继续持续 $t$ 时间的概率，与它刚开始时持续 $t$ 时间的概率是相同的！这是指数分布独有的**"特异功能"**！
        *   **示例应用**：若电子元件寿命 $X \sim Exp(0.1)$，已知该元件已使用 1.5 年，求它还能使用两年（即 $P(X > 1.5+2 \,|\, X > 1.5)$）的概率。根据无记忆性，这个概率就等于 $P(X > 2) = e^{-0.1 \times 2} = e^{-0.2}$。

#### **(3) 伽马分布 $X \sim \Gamma(\alpha, \lambda)$
(拓展知识点，与指数分布关联紧密！)
*   **概念**：伽马分布是指数分布的推广，通常用于描述等待一系列独立事件发生所需的时间。
*   **伽马函数 $\Gamma(\alpha)$**：
    *   定义：$\Gamma(\alpha) = \int_0^{\infty} x^{\alpha-1}e^{-x} \, dx$
    *   性质：
        *   $\Gamma(1) = 1$
        *   $\Gamma(1/2) = \sqrt{\pi}$
        *   $\Gamma(\alpha+1) = \alpha\Gamma(\alpha)$
        *   当 $n$ 为正整数时，$\Gamma(n) = (n-1)!$
*   **参数**：$\alpha > 0$ (形状参数), $\lambda > 0$ (尺度参数/速率参数)。
*   **概率密度函数 (PDF)**：
    $$ f(x) = \begin{cases}
    \frac{\lambda^\alpha}{\Gamma(\alpha)} x^{\alpha-1}e^{-\lambda x}, & x > 0 \\
    0, & x \le 0
    \end{cases} $$
*   **与指数分布的关系**：当 $\alpha = 1$ 时，伽马分布 $\Gamma(1, \lambda)$ 就是指数分布 $Exp(\lambda)$。
    *   你可以把伽马分布看作是**多个独立同分布指数随机变量之和**的分布（当 $\alpha$ 为正整数时）。

#### **(4) 正态分布 $X \sim N(\mu, \sigma^2)$ (最最重要的分布，没有之一！考研必考，反复考！)**
*   **参数**：$\mu$ (均值), $\sigma^2$ (方差)。$\sigma > 0$ 是标准差。
*   **概率密度函数 (PDF)**：
    $$ f(x) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x - \mu)^2}{2\sigma^2}}, \quad -\infty < x < +\infty $$
*   **数字特征**：
    *   **期望 $E(X) = \mu$**
    *   **方差 $D(X) = \sigma^2$**

*   **标准正态分布 $N(0, 1)$**：
    *   当 $\mu=0, \sigma=1$ 时，正态分布称为标准正态分布，其密度函数记为 $\phi(x)$，分布函数记为 $\Phi(x)$。
    $$\phi(x)= \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{x ^2}{2}}, \quad\Phi(x)=\frac{1}{\sqrt{2\pi}}\int_{-\infty}^{x}e^{-\frac{t^2}{2}}dt$$
    *   **概率计算的关键**：任意正态分布 $X \sim N(\mu, \sigma^2)$ 都可以通过标准化变换 $Z = \frac{X-\mu}{\sigma}$ 转化为标准正态分布 $Z \sim N(0,1)$。
    *   **$\Phi(x)$ 的性质**：
        *   $\Phi(0) = 0.5$ (因为对称性)。
        *   $\Phi(-x) = 1 - \Phi(x)$ (因为 $\phi(x)$ 关于 $0$ 对称，分布函数也满足这种互补对称性)。
        *   $P\{X \le x\} = \Phi\left(\frac{x-\mu}{\sigma}\right)$。
        *   $P\{|X - \mu| \le k\sigma\} = P\{|Z| \le k\} = \Phi(k) - \Phi(-k) = 2\Phi(k) - 1$。
        * $\Phi(x)$ 不是初等函数，其值需查标准正态分布表，但上述计算性质要熟练应用！

*   **核心性质 (图形特点和参数影响)：**
    *   **图形特点（钟形曲线）**：![[02 一维随机变量及其分布-1763191333052.png|250]]
        *   **对称性**：密度函数曲线关于直线 $x = \mu$ 对称。
        *   **最大值**：在 $x = \mu$ 处达到最大值 $\frac{1}{\sqrt{2\pi}\sigma}$，这是曲线的唯一峰值。
        *   **拐点**：曲线在 $x = \mu \pm \sigma$ 处有拐点。
        *   **渐近线**：当 $|x| \to \infty$ 时，$f(x) \to 0$，即 $x$ 轴是其水平渐近线。
        *   **呈单峰状**。
    *   **参数的影响 (图形形状)**：![[02 一维随机变量及其分布-1763191474760.png|400]]
        *   **$\mu$ 决定了图的中心位置**：改变 $\mu$ 只会使曲线沿 $x$ 轴平移，形状不变。
        *   **$\sigma$ 决定了图的"胖瘦"**（离散程度）：
            *   $\sigma$ 越大，曲线越扁平，越分散（"胖"）。
            *   $\sigma$ 越小，曲线越高耸，越集中（"瘦"）。

    *   **线性组合的正态性 (考研高频考点！)**：
        *   若 $X_1, X_2, \ldots, X_n$ 是相互独立的随机变量，且 $X_i \sim N(\mu_i, \sigma_i^2)$，则它们的线性组合 $Y = c_1X_1 + c_2X_2 + \ldots + c_nX_n + C$ 仍然服从正态分布。
        *   其新的期望为 $E(Y) = c_1\mu_1 + \ldots + c_n\mu_n + C$。
        *   其新的方差为 $D(Y) = c_1^2\sigma_1^2 + \ldots + c_n^2\sigma_n^2$。(注意系数的平方！)
        *  **✨ 两个相互独立的（或不相关的）正态分布之和：仍是正态分布！✨$X+Y \sim N(\mu_X + \mu_Y, \sigma_X^2 + \sigma_Y^2)$
            1.  **设 $X$ 服从正态分布 $N(\mu_X, \sigma_X^2)$**
                *   均值 $E(X) = \mu_X$
                *   方差 $D(X) = \sigma_X^2$
            2.  **设 $Y$ 服从正态分布 $N(\mu_Y, \sigma_Y^2)$**
                *   均值 $E(Y) = \mu_Y$
                *   方差 $D(Y) = \sigma_Y^2$
            3.  **关键条件**：$X$ 和 $Y$ **相互独立**（或者对于正态分布，只需**不相关**即可保证独立）。
            4.  **结论**：那么，它们的和 $Z = X+Y$ 也服从正态分布，其参数为：
                *   **新均值**：$E(Z) = E(X+Y) = E(X) + E(Y) = \mu_X + \mu_Y$
                *   **新方差**：$D(Z) = D(X+Y) = D(X) + D(Y) = \sigma_X^2 + \sigma_Y^2$ （注意：这里方差可以直接相加，正是因为 $X$ 和 $Y$ **独立**！）

## 02-4 随机变量的函数分布

### 1. 离散型 → 离散型
* 若$Y = g(X)$，且$X$是离散型随机变量，则$Y$也是离散型。
* **方法**：
    1. 列出$X$的所有可能取值$x_i$及其概率$P(X=x_i) = p_i$。
    2. 计算$Y$的所有可能取值$y_j = g(x_i)$。
    3. 将所有导致相同$y_j$的$p_i$相加，得到$P(Y=y_j) = \sum_{x_i: g(x_i)=y_j} P(X=x_i)$。

### 2. 连续型 → 连续型（或混合型）(考研重点和难点！)
* 若$X$是连续型随机变量，且$Y = g(X)$，求$Y$的分布。
* **通用方法：分布函数法 (万能法)**：
    1. **确定 $Y$ 的取值范围**：根据 $X$ 的取值范围和 $g(X)$ 的函数形式。
    2. **计算 $Y$ 的分布函数 $F_Y(y)$**：
        $$ F_Y(y) = P(Y \leq y) = P(g(X) \leq y) $$
        * **关键**：将不等式 $g(X) \leq y$ 转换为关于 $X$ 的不等式（或区间），然后利用 $X$ 的已知分布函数 $F_X(x)$ 或概率密度函数 $f_X(x)$ 进行积分计算。
    3. **求导得到 $Y$ 的概率密度函数 $f_Y(y)$**：
        * $f_Y(y) = F_Y'(y)$ (在 $f_Y(y)$ 连续的区间内)。
* **特殊方法：变量替换法 (针对单调可导函数)**：
    * 若 $g(x)$ 是**严格单调可导函数**，且其反函数 $x = h(y)$ 存在，那么 $Y$ 的概率密度函数为：
        $$ f_Y(y) = f_X(h(y)) |h'(y)| $$
        * **注意**：此公式仅在 $Y$ 的有效取值范围内使用，其余地方 $f_Y(y) = 0$。
* **多维随机变量函数的分布 (例如 $Z = g(X, Y)$)**：
    * 这是更复杂的情况，**考研中常考**，特别是涉及**二维随机变量函数的分布**，如 $Z = X+Y$, $Z=X-Y$, $Z=XY$, $Z=X/Y$ 等。
    * **通用方法仍然是分布函数法**：
        1. **确定 $Z$ 的取值范围**。
        2. **计算 $Z$ 的分布函数 $F_Z(z) = P(Z \leq z) = P(g(X, Y) \leq z)$**。
            * **关键**：将不等式 $g(X, Y) \leq z$ 转换为 $xy$-平面上的一个区域，然后利用 $(X, Y)$ 的**联合概率密度函数 $f(x, y)$** 在该区域上进行**二重积分**。
                $$ F_Z(z) = \iint_{g(x,y) \le z} f(x, y) \, dx \, dy $$
            * **独立性**：如果 $X, Y$ 独立，则 $f(x, y) = f_X(x)f_Y(y)$，这会大大简化积分。
        3. **求导得到 $Z$ 的概率密度函数 $f_Z(z)$**。

    * **特殊方法：卷积公式 (针对和 $Z = X+Y$)**：
        * 如果 $X$ 和 $Y$ 相互独立，且概率密度函数分别为 $f_X(x)$ 和 $f_Y(y)$，那么它们的和 $Z = X+Y$ 的概率密度函数为：
            $$ f_Z(z) = \int_{-\infty}^{\infty} f_X(x) f_Y(z-x) \, dx \quad \text{或} \quad \int_{-\infty}^{\infty} f_X(z-y) f_Y(y) \, dy $$
        * **注意**：积分限需要根据 $f_X$ 和 $f_Y$ 的定义域来确定。
