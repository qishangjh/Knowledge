# 01 随机事件与概率
## **01-1 基本概念**

### 1. 随机试验（Random Experiment）
- **定义**：在相同条件下可以重复进行；每次试验的结果不止一个；在试验前不能确定哪个结果会发生。
- **特点**：
  - 可重复性
  - 结果不确定性
  - 所有可能结果已知

### 2. 随机事件（Random Event）
- **定义**：随机试验中可能发生也可能不发生的结果，是样本空间的子集。
- **表示**：用大写字母 A、B、C 等表示。

### 3. 样本空间（Sample Space）
- **定义**：一个随机试验所有可能结果的集合，记为 **Ω**。
- **基本事件**：样本空间中的每一个元素，即一个最简单的结果。

### 4. 事件的关系与运算

#### （1）关系
| 关系类型 | 符号 | 含义 |
|----------|------|------|
| 包含     | A ⊆ B | 若 A 发生，则 B 一定发生 |
| 相等     | A = B | A 和 B 包含相同的样本点 |
| 积（交）| A ∩ B | A 和 B 同时发生 |
| 相容     | A ∩ B ≠ ∅ | A 和 B 有可能同时发生 |
| 互斥     | A ∩ B = ∅ | A 和 B 不能同时发生 |
| 和（并）| A ∪ B | A 或 B 至少有一个发生 |
| 差       | A - B | A 发生而 B 不发生 |
| 逆（对立）| A<sup>c</sup> 或 Ā | 表示 A 不发生，即 Ω - A |

#### （2）运算律
| 名称 | 内容 |
|------|------|
| 吸收律 | A ∪ (A ∩ B) = A，A ∩ (A ∪ B) = A |
| 交换律 | A ∪ B = B ∪ A，A ∩ B = B ∩ A |
| 结合律 | (A ∪ B) ∪ C = A ∪ (B ∪ C)，(A ∩ B) ∩ C = A ∩ (B ∩ C) |
| 分配律 | A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)，A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C) |
| 对偶律（De Morgan 定律）| (A ∪ B)<sup>c</sup> = A<sup>c</sup> ∩ B<sup>c</sup>，(A ∩ B)<sup>c</sup> = A<sup>c</sup> ∪ B<sup>c</sup> |

### 5. 概率的定义

#### （1）描述性定义
- 概率是对事件发生可能性大小的度量，通常用一个介于 0 与 1 之间的数表示。

#### （2）统计性定义
- 在大量重复试验中，事件 A 发生的频率稳定地接近某个常数，这个常数称为事件 A 的概率，记作 P(A)。

#### （3）公式化定义（公理化定义）
设样本空间为 Ω，对每个事件 A 赋予一个实数 P(A)，满足以下三条公理：

1. **非负性**：P(A) ≥ 0
2. **规范性**：P(Ω) = 1
3. **可列可加性**：若 A₁, A₂, … 互不相容，则
   $$
   P\left(\bigcup_{i=1}^{\infty} A_i\right) = \sum_{i=1}^{\infty} P(A_i)
   $$

## **01-2 古典概型和几何概型**

### 1. 古典概型（Classical Probability Model）

#### 特点
- 有限性：样本空间中只有有限个样本点
- 等可能性：每个样本点出现的可能性相同

#### 概率公式
$$
P(A) = \frac{\text{有利事件数}}{\text{总事件数}} = \frac{|A|}{|\Omega|}
$$

#### 典型问题类型
##### （1）随机分配问题
- 如：将 n 个球放入 m 个盒子中，每个球独立分配，求某种分配方式的概率。
##### （2）简单随机抽样问题
- 如：从 N 个个体中无放回地抽取 n 个个体，求某个特定组合被选中的概率。

计算方法：
1. 列举法（直接数数法）
2. 集合对应法
    1. 加法原理
    2. 乘法原理
    3. 排列：从n个不同的元素中取出m(<=n)个元素，并按照一定顺序排成一列，所有排列的个数：$P_n^m=n(n-1)(n-2)…(n-m+1)=\frac{n!}{(n-m)!}$,当m=n时，$P^n_n=\frac{n!}{0!}$，叫做全排列。
    4. 组合：从n个不同的元素中取出m个元素，并组成一组，所有组合数：$C_n^m=\frac{P^m_n}{m!}$
3. 逆数法
---

### 2. 几何概型（Geometric Probability Model）

#### 特点
- 无限性：样本空间为无限连续区域（如线段、平面区域、空间区域等）
- 等可能性：每个位置出现的可能性相同

#### 概率公式
$$
P(A) = \frac{\text{区域 A 的测度（长度、面积、体积等）}}{\text{样本空间 Ω 的测度}}
$$

#### 应用举例
- 蒲丰投针问题
- 在单位正方形内随机取点，求点落在某个子区域内的概率

## 01-3 概率的性质与公式

### 1. 性质

#### 1. 有界性
- **非负性**：对于任何事件 $A$，其概率 $P(A) \geq 0$。
- **规范性**：必然事件 $\Omega$ (或 $S$) 的概率为 1，即 $P(\Omega) = 1$。不可能事件 $\emptyset$ 的概率为 0，即 $P(\emptyset) = 0$。
- **最大值**：任何事件的概率都不会超过 1，即 $P(A) \leq 1$。
    - 综合起来：$0 \leq P(A) \leq 1$。

#### 2. 单调性
如果事件 $A$ 包含事件 $B$ (即 $B \subset A$)，那么 $P(B) \leq P(A)$。
- **推论**：如果 $A$ 和 $B$ 是两个事件，则 $P(A - B) = P(A) - P(A \cap B)$。

### 2. 公式

#### 1. 逆事件概率公式
- **概念**：事件 $A$ 的逆事件（或对立事件）记作 $\bar{A}$，表示 $A$ 不发生。
- **公式**：$P(\bar{A}) = 1 - P(A)$。
- **应用**：当直接计算事件 $A$ 的概率比较困难时，可以考虑计算其逆事件 $\bar{A}$ 的概率，然后用 1 减去。

#### 2. 加法公式
- **概念**：计算两个或多个事件中至少有一个发生的概率。
- **公式**：
    - 对于任意两个事件 $A$ 和 $B$：
        $$ P(A \cup B) = P(A) + P(B) - P(A \cap B) $$
    - 如果 $A$ 和 $B$ 是**互斥事件**（即 $A \cap B = \emptyset$），则 $P(A \cup B) = P(A) + P(B)$。
    - 对于任意三个事件 $A, B, C$：
        $$ P(A \cup B \cup C) = P(A) + P(B) + P(C) - P(A \cap B) - P(A \cap C) - P(B \cap C) + P(A \cap B \cap C) $$
- **应用**：计算"或"事件的概率。

#### 3. 减法公式
- **概念**：计算事件 $A$ 发生但事件 $B$ 不发生的概率。
- **公式**：$P(A - B) = P(A) - P(A \cap B)$。
- **推论**：如果 $B \subset A$，则 $P(A - B) = P(A) - P(B)$。

#### 4. 条件概率公式
- **概念**：在事件 $B$ 已经发生的条件下，事件 $A$ 发生的概率。
- **公式**：
    $$ P(A \mid B) = \frac{P(A \cap B)}{P(B)} \quad (P(B) > 0) $$
- **乘法公式**：由条件概率公式变形得到，用于计算交事件的概率：
    $$ P(A \cap B) = P(B) P(A \mid B) = P(A) P(B \mid A) $$
    对于多个事件：
    $$ P(A_1 \cap A_2 \cap \cdots \cap A_n) = P(A_1) P(A_2 \mid A_1) P(A_3 \mid A_1 \cap A_2) \cdots P(A_n \mid A_1 \cap \cdots \cap A_{n-1}) $$
- **应用**：处理事件之间有依赖关系的情况。

#### 5. 全概率公式

- **概念**：用于计算一个事件 $A$ 的概率，当事件 $A$ 的发生依赖于一系列互斥且构成完备事件组的事件 $B_i$ 时。
- **应用**：将复杂事件的概率分解为在不同条件下发生的概率之和。
- **前提条件**：
假设样本空间 $S$ 被事件 $B_1, B_2, \dots, B_n$ 构成了一个划分（partition），即：
1.  **互斥性：** 任意两个事件 $B_i$ 和 $B_j$ 互斥，即 $B_i \cap B_j = \emptyset$ (当 $i \neq j$)。
2.  **穷尽性：** 所有事件 $B_i$ 的并集等于样本空间 $S$，即 $B_1 \cup B_2 \cup \dots \cup B_n = S$。
3.  **非零概率：** 每个 $P(B_i) > 0$。

- **公式**：设 $B_1, B_2, \ldots, B_n$ 是一个完备事件组（即它们两两互斥，且并集为必然事件），且 $P(B_i) > 0$，则对于任意事件 $A$，有：
    $$ P(A) = \sum_{i=1}^n P(A \mid B_i) P(B_i) $$
$$P(A) = P(A|B_1)P(B_1) + P(A|B_2)P(B_2) + \dots + P(A|B_n)P(B_n)$$

-   $P(A)$: 事件 $A$ 发生的总概率。
-   $P(B_i)$: 事件 $B_i$ 发生的先验概率。
-   $P(A|B_i)$: 在事件 $B_i$ 发生的条件下，事件 $A$ 发生的条件概率。

#### 6. 贝叶斯公式 (Bayes' Formula / 逆概率公式)
- **概念**：在事件 $A$ 已经发生的条件下，计算导致 $A$ 发生的某个原因事件 $B_k$ 的概率。它将"先验概率"转化为"后验概率"。
- **公式**：设 $B_1, B_2, \ldots, B_n$ 是一个完备事件组，且 $P(B_i) > 0$，则对于任意事件 $A$ (且 $P(A) > 0$)，有：
    $$ P(B_k \mid A) = \frac{P(A \mid B_k) P(B_k)}{\sum_{i=1}^n P(A \mid B_i) P(B_i)} $$
- **应用**：在已知结果的情况下，推断原因的概率，常用于诊断、模式识别等领域。

## 01-4 概率的独立性和独立重复试验

### 1. 事件的独立性
- **概念**：如果事件 $A$ 的发生与否不影响事件 $B$ 发生的概率，则称事件 $A$ 和 $B$ **相互独立**。
- **定义**：事件 $A$ 和 $B$ 相互独立，当且仅当：
    $$ P(A \cap B) = P(A) P(B) $$
- **推广到多个事件**：对于 $n$ 个事件 $A_1, A_2, \ldots, A_n$，如果对于其中任意 $k$ 个事件 $(1 \leq k \leq n)$，它们的交事件的概率都等于各自概率的乘积，则称这 $n$ 个事件**相互独立**。
    - 例如，对于三个事件 $A, B, C$ 相互独立，需要满足：
        - $P(A \cap B) = P(A)P(B)$
        - $P(A \cap C) = P(A)P(C)$
        - $P(B \cap C) = P(B)P(C)$
        - $P(A \cap B \cap C) = P(A)P(B)P(C)$
- **重要性质**：如果 $A$ 和 $B$ 相互独立，那么 $A$ 与 $\bar{B}$，$\bar{A}$ 与 $B$，$\bar{A}$ 与 $\bar{B}$ 也都相互独立。

### 2. 独立性的判定
- **根据定义**：计算 $P(A \cap B)$ 和 $P(A)P(B)$，看它们是否相等。
- **根据条件概率**：如果 $P(A \mid B) = P(A)$ (且 $P(B) > 0$)，则 $A$ 和 $B$ 相互独立。

### 3. 独立实验序列概型与 $n$ 重伯努利概型

#### 1. 独立实验序列概型
- **概念**：如果一系列试验的结果互不影响，则称这些试验是**独立试验**。
- **应用**：计算一系列独立事件同时发生的概率，只需将各自的概率相乘。

#### 2. $n$ 重伯努利概型 (n-fold Bernoulli Trials)
- **概念**：指在相同条件下独立重复进行 $n$ 次试验，每次试验只有两种可能结果：成功（概率为 $p$）或失败（概率为 $1-p$）。
- **二项分布**：在 $n$ 重伯努利试验中，成功次数 $X$ 服从二项分布 $B(n, p)$，其概率分布律为：
    $$ P(X = k) = C_n^k p^k (1 - p)^{n-k}, \quad k = 0, 1, \ldots, n $$
- **应用**：广泛用于描述重复试验中成功次数的概率，如产品合格率、抛硬币等。
