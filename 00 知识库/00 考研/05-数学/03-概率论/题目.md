# 题目

## 第18题：均匀分布参数估计的"精打细算"

**题目**：设总体 $X$ 服从 $[0, \theta]$ 上的均匀分布，其中 $\theta \in (0, +\infty)$ 为未知参数。$X_1, X_2, \ldots, X_n$ 是来自总体 $X$ 的简单随机样本。  
记 $X_{(n)} = \max\{X_1, X_2, \ldots, X_n\}$, $T_c = cX_{(n)}$。  
(1) 求 $c$，使得 $E(T_c) = \theta$；  
(2) 记 $h(c) = E[(T_c - \theta)^2]$，求 $c$ 使得 $h(c)$ 最小。

### **核心思路分解**

1. **求出顺序统计量 $X_{(n)}$ 的分布**：这是后续计算期望和方差的基础。
2. **计算 $E(X_{(n)})$ 和 $E(X_{(n)}^2)$**：这些是计算 $E(T_c)$ 和 $h(c)$ 所需的中间量。
3. **Part (1)**：利用 $E(T_c) = \theta$ 求解 $c$，即求 $\theta$ 的**无偏估计量**。
4. **Part (2)**：利用均方误差（MSE）的定义 $h(c) = E[(T_c - \theta)^2]$，通过求导找到使 $h(c)$ 最小的 $c$。

---

### **第一步：确定 $X_{(n)}$ 的分布**

* **总体 $X$ 的分布**：$X \sim U[0, \theta]$。
    * 概率密度函数 (PDF)：$f_X(x) = \begin{cases} 1/\theta, & 0 \le x \le \theta \\ 0, & \text{其他} \end{cases}$。
    * 分布函数 (CDF)：$F_X(x) = \begin{cases} 0, & x < 0 \\ x/\theta, & 0 \le x < \theta \\ 1, & x \ge \theta \end{cases}$。
* **顺序统计量 $X_{(n)} = \max\{X_1, \ldots, X_n\}$ 的分布函数**：  
    $F_{X_{(n)}}(x) = P(X_{(n)} \le x) = P(X_1 \le x, \ldots, X_n \le x)$。  
    由于 $X_i$ 是独立同分布的，所以 $P(X_1 \le x, \ldots, X_n \le x) = [P(X \le x)]^n = [F_X(x)]^n$。  
    因此，$F_{X_{(n)}}(x) = \begin{cases} 0, & x < 0 \\ (x/\theta)^n, & 0 \le x < \theta \\ 1, & x \ge \theta \end{cases}$。

* **顺序统计量 $X_{(n)}$ 的概率密度函数**：  
    $f_{X_{(n)}}(x) = F'_{X_{(n)}}(x)$。  
    当 $0 \le x < \theta$ 时，$f_{X_{(n)}}(x) = \frac{\mathrm{d}}{\mathrm{d}x} \left( \frac{x}{\theta} \right)^n = n \frac{x^{n-1}}{\theta^n}$。  
    所以，$f_{X_{(n)}}(x) = \begin{cases} n x^{n-1}/\theta^n, & 0 \le x < \theta \\ 0, & \text{其他} \end{cases}$。

### **第二步：计算 $E(X_{(n)})$ 和 $E(X_{(n)}^2)$**

* **计算 $E(X_{(n)})$**：  
    $E(X_{(n)}) = \int_0^\theta x \cdot f_{X_{(n)}}(x) \, \mathrm{d}x = \int_0^\theta x \cdot \frac{n x^{n-1}}{\theta^n} \, \mathrm{d}x = \frac{n}{\theta^n} \int_0^\theta x^n \, \mathrm{d}x$  
    $E(X_{(n)}) = \frac{n}{\theta^n} \left[ \frac{x^{n+1}}{n+1} \right]_0^\theta = \frac{n}{\theta^n} \frac{\theta^{n+1}}{n+1} = \frac{n\theta}{n+1}$。

* **计算 $E(X_{(n)}^2)$**：  
    $E(X_{(n)}^2) = \int_0^\theta x^2 \cdot f_{X_{(n)}}(x) \, \mathrm{d}x = \int_0^\theta x^2 \cdot \frac{n x^{n-1}}{\theta^n} \, \mathrm{d}x = \frac{n}{\theta^n} \int_0^\theta x^{n+1} \, \mathrm{d}x$  
    $E(X_{(n)}^2) = \frac{n}{\theta^n} \left[ \frac{x^{n+2}}{n+2} \right]_0^\theta = \frac{n}{\theta^n} \frac{\theta^{n+2}}{n+2} = \frac{n\theta^2}{n+2}$。

---

### **Part (1): 求 $c$，使得 $E(T_c) = \theta$**

* 我们有 $T_c = cX_{(n)}$。
* 根据期望的线性性质：$E(T_c) = E(cX_{(n)}) = cE(X_{(n)})$。
* 将 $E(X_{(n)})$ 的结果代入：$E(T_c) = c \cdot \frac{n\theta}{n+1}$。
* 根据题目要求 $E(T_c) = \theta$：  
    $c \frac{n\theta}{n+1} = \theta$
* 由于 $\theta \in (0, +\infty)$，所以 $\theta \ne 0$，可以两边同除以 $\theta$：  
    $c \frac{n}{n+1} = 1$  
    $c = \frac{n+1}{n}$。

**Part (1) 结果**：$c = \frac{n+1}{n}$。  
这个 $T_c = \frac{n+1}{n} X_{(n)}$ 是 $\theta$ 的**无偏估计量**。

---

### **Part (2): 记 $h(c) = E[(T_c - \theta)^2]$，求 $c$ 使得 $h(c)$ 最小**

* $h(c)$ 是估计量 $T_c$ 的**均方误差 (MSE)**。我们知道 $MSE(T_c) = D(T_c) + [Bias(T_c)]^2$。

1. **计算 $D(T_c)$**：
    * 首先计算 $D(X_{(n)})$：  
        $D(X_{(n)}) = E(X_{(n)}^2) - [E(X_{(n)})]^2 = \frac{n\theta^2}{n+2} - \left( \frac{n\theta}{n+1} \right)^2$  
        $D(X_{(n)}) = n\theta^2 \left( \frac{1}{n+2} - \frac{n}{(n+1)^2} \right) = n\theta^2 \frac{(n+1)^2 - n(n+2)}{(n+2)(n+1)^2}$  
        $D(X_{(n)}) = n\theta^2 \frac{(n^2 + 2n + 1) - (n^2 + 2n)}{(n+2)(n+1)^2} = n\theta^2 \frac{1}{(n+1)^2(n+2)}$。
    * 然后计算 $D(T_c)$：  
        $D(T_c) = D(cX_{(n)}) = c^2 D(X_{(n)}) = c^2 \frac{n\theta^2}{(n+1)^2(n+2)}$。

2. **计算 $Bias(T_c)$**：  
    $Bias(T_c) = E(T_c) - \theta = c \frac{n\theta}{n+1} - \theta = \theta \left( \frac{cn}{n+1} - 1 \right)$。

3. **构建 $h(c)$ 的表达式**：  
    $h(c) = E[(T_c - \theta)^2] = D(T_c) + [Bias(T_c)]^2$  
    $h(c) = c^2 \frac{n\theta^2}{(n+1)^2(n+2)} + \theta^2 \left( \frac{cn}{n+1} - 1 \right)^2$  
    可以提取 $\theta^2$ 作为公因子：  
    $h(c) = \theta^2 \left[ c^2 \frac{n}{(n+1)^2(n+2)} + \left( \frac{cn}{n+1} - 1 \right)^2 \right]$。

4. **对 $h(c)$ 求导，令导数为 $0$ 求解 $c$**：  
    由于 $\theta^2 > 0$，我们只需要对括号内的部分求导并令其为 $0$。  
    令 $g(c) = c^2 \frac{n}{(n+1)^2(n+2)} + \left( \frac{cn}{n+1} - 1 \right)^2$。  
    $g'(c) = 2c \frac{n}{(n+1)^2(n+2)} + 2 \left( \frac{cn}{n+1} - 1 \right) \left( \frac{n}{n+1} \right)$。  
    令 $g'(c) = 0$，并同除以 $2n/(n+1)$：  
    $c \frac{1}{(n+1)(n+2)} + \left( \frac{cn}{n+1} - 1 \right) = 0$。  
    $c \frac{1}{(n+1)(n+2)} + \frac{cn}{n+1} - 1 = 0$。  
    同乘以 $(n+1)(n+2)$ 消除分母：  
    $c + cn(n+2) - (n+1)(n+2) = 0$  
    $c [1 + n(n+2)] = (n+1)(n+2)$  
    $c [1 + n^2 + 2n] = (n+1)(n+2)$  
    $c (n+1)^2 = (n+1)(n+2)$  
    由于 $n \ge 1$，所以 $n+1 \ne 0$，可以两边同除以 $(n+1)$：  
    $c = \frac{n+2}{n+1}$。

通过二阶导数判别，可以知道这是最小值点（或者注意到 $h(c)$ 是一个开口向上的二次函数）。

**Part (2) 结果**：$c = \frac{n+2}{n+1}$。  
这个 $c$ 值使得 $T_c = \frac{n+2}{n+1} X_{(n)}$ 的均方误差最小。

---

**小登小课堂**：

* **顺序统计量 $X_{(n)}$**：对于均匀分布，最大值 $X_{(n)}$ 的分布函数和密度函数推导是基础。
* **无偏估计**：$E(\hat{\theta}) = \theta$，这是估计量的一个重要性质。Part (1) 就是在找无偏估计。
* **均方误差 (MSE)**：$MSE(\hat{\theta}) = E[(\hat{\theta} - \theta)^2] = D(\hat{\theta}) + [Bias(\hat{\theta})]^2$。这是评估估计量好坏的重要标准，它综合考虑了估计量的**精度**（方差）和**准确性**（偏差）。Part (2) 就是在找使 MSE 最小的估计量。
* **MSE 最优估计量与无偏估计量**：通常情况下，使得 MSE 最小的估计量不一定是无偏的，反之亦然。在本题中，使 $E(T_c) = \theta$ 的 $c = \frac{n+1}{n}$，而使 $h(c)$ 最小的 $c = \frac{n+2}{n+1}$，两者**不同**，这体现了均方误差和无偏性的权衡。
