# 15 微分方程

## 15-1 微分方程的基本概念
### 1. 微分方程及其阶
* **定义**：含有未知函数及其导数（或微分）的方程，称为微分方程。
    示例：$y'' + 2y' + y = 0$（含二阶导数）、$x^2 dy + y dx = 0$（含一阶微分）。
* **阶的判定**：方程中出现的**未知函数的最高阶导数的阶数**，即为微分方程的阶。
    示例：$y''' - 3y' + 2y = x$ 是三阶微分方程，$y' = \sin x$ 是一阶微分方程。

### 2. 常微分方程
* **定义**：未知函数为**一元函数**，仅含常导数（非偏导数）的微分方程。
* **与偏微分方程的区别**：偏微分方程的未知函数是多元函数，含偏导数（如 $\frac{\partial u}{\partial x} + \frac{\partial u}{\partial y} = 0$），本章节重点讨论常微分方程。

### 3. 线性微分方程
* **n 阶线性微分方程标准形式**：
    $y^{(n)} + a_1(x)y^{(n-1)} + \dots + a_{n-1}(x)y' + a_n(x)y = f(x)$
若右端函数/(x)恒为零，则称方程为 n阶 #齐次线性微分方程 ，否则称其为 n阶 #非齐次线性微分方程
*   **齐次方程**：右端项为零，通常表示系统在没有外部激励下的自然响应。
        $$ y'' + P(x)y' + Q(x)y = 0 \quad (1) $$
    *   **非齐次方程**：右端项 $f(x)$ 不为零，表示系统受到外部激励或输入。
        $$ y'' + P(x)y' + Q(x)y = f(x) \quad (2) $$
        其中 $f(x)$ 称为**非齐次项**。
* **核心特点**：
    * 未知函数 $y$ 及其各阶导数均为**一次项**（无平方、乘积等非线性项）；
    * 系数 $a_1(x), a_2(x), \dots, a_n(x)$ 仅与自变量 $x$ 有关，不含 $y$ 或其导数。
        示例：$y'' + xy' + 2y = e^x$（线性）、$y'' + y \cdot y' = 0$（非线性，含 $y \cdot y'$ 乘积项）。

### 4. 微分方程的通解与特解
微分方程解的图形成为 #积分曲线

| 类型  | 定义                                 | 示例                                                    |
| --- | ---------------------------------- | ----------------------------------------------------- |
| #通解 | 含有**独立任意常数**，且常数个数等于方程阶数的解（解的一般形式）| 一阶方程 $y' = 2x$ 的通解为 $y = x^2 + C$（含 1 个常数 $C$）|
| #特解 | 确定了通解中所有任意常数具体值后的解                 | 若给通解 $y = x^2 + C$ 附加条件 $y(0) = 1$，则特解为 $y = x^2 + 1$ |

### 5. 初始条件与特解
* **初始条件**：用于确定通解中任意常数的条件，通常是未知函数在某点的函数值或导数值，条件个数与方程阶数一致。
    示例：
    * 一阶方程：初始条件为 $y(x_0) = y_0$（1 个条件）；
    * 二阶方程：初始条件为 $y(x_0) = y_0$ 且 $y'(x_0) = y_1$（2 个条件）。
* **关系**：通解 + 初始条件 $\implies$ 特解（唯一确定）。

### 1. 可分离变量型微分方程
* **标准形式**：能整理为 $f(x)dx = g(y)dy$（变量 $x$、$y$ 分别在等号两侧，无交叉项）。
* **求解步骤**：
1. 分离变量：将方程化为 $f(x)dx = g(y)dy$；
2. 两边积分：$\int f(x)dx = \int g(y)dy + C$（$C$ 为任意常数）；
3. 整理结果：得到通解（隐式或显式形式）。
    示例：求解 $y' = xy$，分离变量得 $\frac{1}{y}dy = xdx$，积分得 $\ln|y| = \frac{1}{2}x^2 + C$，整理为 $y = Ce^{\frac{1}{2}x^2}$。
> [!warning] 积分注意事项
> 1. 开绝对值加正负
> 2. 合并所有可变为常数的部分
> 3. 对于$\int\frac{1}{x}dx=\ln|x|+c$两种情况不加绝对值
>     1. 题目已知x为正
>     2. 线性方程求通解时：$e^{-\int pax}$与$e^{\int pax}$

---

### 2. 齐次型微分方程
* **标准形式**：$\frac{dy}{dx} = \varphi\left(\frac{y}{x}\right)$（方程右边仅为 $\frac{y}{x}$ 的函数，无单独的 $x$ 或 $y$ 项）。
* **求解步骤（变量替换法）**：
1. 令 $u = \frac{y}{x}$（即 $y = ux$），则 $\frac{dy}{dx} = u + x\frac{du}{dx}$（复合函数求导）；
2. 代入原方程：$u + x\frac{du}{dx} = \varphi(u)$，整理为可分离变量型 $\frac{du}{\varphi(u) - u} = \frac{dx}{x}$；
3. 积分求解：按可分离变量方程求解，得到 $u$ 与 $x$ 的关系；
4. 回代变量：将 $u = \frac{y}{x}$ 代入，得到 $y$ 与 $x$ 的通解。

### 3. 一阶线性微分方程
* **标准形式**：$\frac{dy}{dx} + P(x)y = Q(x)$（线性结构，$y$ 和 $y'$ 为一次项，系数 $P(x)$、非齐次项 $Q(x)$ 仅与 $x$ 有关）。
* **通解公式**（常数变易法推导）：
    $y = e^{-\int P(x)dx} \left( \int Q(x) e^{\int P(x)dx} dx + C \right)$

* **求解步骤**：
1. 确定 $P(x)$ 和 $Q(x)$（对照标准形式，注意符号）；
2. 计算积分因子 $e^{\int P(x)dx}$；
3. 代入通解公式，计算积分并整理结果。
    示例：求解 $y' + \frac{1}{x}y = x$，$P(x) = \frac{1}{x}$，$Q(x) = x$，积分因子 $e^{\int \frac{1}{x}dx} = x$，通解为 $y = \frac{1}{x}\left( \int x \cdot x dx + C \right) = \frac{1}{3}x^2 + \frac{C}{x}$。

### 4. 伯努利方程
* **标准形式**：$\frac{dy}{dx} + P(x)y = Q(x)y^n$（$n \neq 0,1$，非线性，但可转化为线性方程）。
* **求解步骤（降次转化）**：
1. 两边除以 $y^n$：$y^{-n}\frac{dy}{dx} + P(x)y^{1-n} = Q(x)$；
2. 令 $z = y^{1-n}$（降次），则 $\frac{dz}{dx} = (1-n)y^{-n}\frac{dy}{dx}$，代入方程得：
    $\frac{dz}{dx} + (1-n)P(x)z = (1-n)Q(x)$
3. 按一阶线性方程求解 $z$，再回代 $z = y^{1-n}$，得到原方程通解。

### 5. 二阶可降阶微分方程（特殊高阶方程）
通过变量替换将二阶方程降为一阶方程求解，按缺项类型分类：

| 类型             | 标准形式             | 求解方法                                                                                      |
| -------------- | ---------------- | ----------------------------------------------------------------------------------------- |
| 缺 $y$ 和 $y'$ 项 | $y'' = f(x)$     | 连续积分两次：$y' = \int f(x)dx + C_1$，$y = \int \left( \int f(x)dx + C_1 \right)dx + C_2$       |
| 缺 $y$ 项        | $y'' = f(x, y')$ | 令 $p = y'$（则 $y'' = \frac{dp}{dx}$），转化为一阶方程 $\frac{dp}{dx} = f(x, p)$，求解 $p$ 后积分得 $y$     |
| 缺 $x$ 项        | $y'' = f(y, y')$ | 令 $p = y'$（则 $y'' = p\frac{dp}{dy}$），转化为一阶方程 $p\frac{dp}{dy} = f(y, p)$，求解 $p$ 后分离变量得 $y$ |
|                | $y^{(n)}=f(x)$   | n阶就积分n次，通解中含有n个不关联常数C                                                                     |

### 6. 全微分方程
![[05-数学01-微积分 15 微分方程 2025-10-23 2547.png]]
* **标准形式**：$P(x,y)dx + Q(x,y)dy = 0$，且满足**全微分条件** $\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$（存在二元函数 $u(x,y)$ 使 $du = Pdx + Qdy$）。
* **通解形式**：$u(x,y) = C$（$C$ 为任意常数），其中 $u(x,y)$ 的计算方法：

    $u(x,y) = \int_{x_0}^x P(x,y_0)dx + \int_{y_0}^y Q(x,y)dy$

（$(x_0,y_0)$ 为使 $P$、$Q$ 连续的任意点，如 $(0,0)$）。

### **Part 1：线性微分方程的基础结构与定理**
1.  **定理 1：齐次方程的通解结构**
    *   **结论**：如果 $y_1(x)$ 和 $y_2(x)$ 是齐次方程 $(1)$ 的两个**线性无关**的特解（也就是它们不是常数倍的关系），那么齐次方程 $(1)$ 的通解就是它们的线性组合：
        $$ Y = C_1 y_1(x) + C_2 y_2(x) $$
        其中 $C_1, C_2$ 是任意常数。
2.  **定理 2：非齐次方程的通解结构**
    *   **结论**：如果 $y^*$ 是非齐次方程 $(2)$ 的一个特解（任何一个都行！），并且 $Y$ 是对应齐次方程 $(1)$ 的通解，那么非齐次方程 $(2)$ 的通解是：
        $$ y = Y + y^* = C_1 y_1(x) + C_2 y_2(x) + y^* $$
3.  **定理 3：非齐次方程特解的性质（差为齐次解）**
    *   **结论**：如果 $y_1^*(x)$ 和 $y_2^*(x)$ 都是非齐次方程 $(2)$ 的特解，那么它们的差 $y_1^*(x) - y_2^*(x)$ **一定是**对应齐次方程 $(1)$ 的解。

4.  **定理 4：非齐次方程特解的线性组合（叠加原理）**
    *   **结论**：如果：
        *   $y_1^*(x)$ 是方程 $y'' + p(x)y' + q(x)y = f_1(x)$ 的一个特解，
        *   $y_2^*(x)$ 是方程 $y'' + p(x)y' + q(x)y = f_2(x)$ 的一个特解，
        *   那么 $y_1^*(x) + y_2^*(x)$ 是方程 $y'' + p(x)y' + q(x)y = f_1(x) + f_2(x)$ 的一个特解。

### **Part 2：二阶常系数线性微分方程的求解**
**方程类型：** $$y'' + py' + qy = f(x)$$
其中 $p, q$ 为常数。

---
#### **I. 齐次线性微分方程 ($f(x) = 0$)**
*   **方程形式：** $$y'' + py' + qy = 0$$
*   **核心方法：** **特征方程法**
    1.  **写出特征方程：** $\lambda^2 + p\lambda + q = 0$
    2.  **求解特征根 $\lambda_1, \lambda_2$：**
    3.  **根据根的类型确定通解 $Y$：**

| **特征根类型**                               | **齐次方程通解 $Y$**                                    | **说明**                                                   |
| :----------------------------------------- | :---------------------------------------------------- | :--------------------------------------------------------- |
| **两个不等实根** $\lambda_1 \neq \lambda_2$ | $$Y = C_1 e^{\lambda_1 x} + C_2 e^{\lambda_2 x}$$     | 两个线性无关的指数函数组合。                               |
| **两个相等实根** $\lambda_1 = \lambda_2 = \lambda$ | $$Y = (C_1 + C_2 x)e^{\lambda x}$$                    | 在指数函数前乘以 $x$ 确保线性无关。                        |
| **一对共轭复根** $\lambda_{1,2} = \alpha \pm i\beta$ | $$Y = e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x)$$ | 复数指数函数通过欧拉公式转换为三角函数形式。             |

#### **II. 非齐次线性微分方程 ($f(x) \neq 0$)**

*   **方程形式：** $$y'' + py' + qy = f(x)$$
*   **通解结构：** $$y = Y + y^*$$
    *   $Y$ 是对应齐次方程 $y'' + py' + qy = 0$ 的通解（按上述方法求）。
    *   $y^*$ 是非齐次方程的一个特解。
*   **核心方法：** **待定系数法**
    1.  **特解 $y^*$ 的设取原则：** 根据非齐次项 $f(x)$ 的形式猜测 $y^*$ 的结构，并引入修正因子 $x^k$。
    2.  **将 $y^*$ 及其各阶导数代入原非齐次方程，利用比较系数法确定待定系数。**

| **非齐次项 $f(x)$ 形式**                                                                         | **特解 $y^*$ 设取原则**                                                                                                                                                                                                                                                          |
| :----------------------------------------------------------------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1. 多项式形式** $P_m(x)$ (m次多项式)                                                              | $$y^* = x^k Q_m(x)$$<br> ($Q_m(x)$ 为 $m$ 次待定多项式)<br> **$k$ 为 $\mathbf{0}$ 作为特征方程根的重数**：<br>  $k=0$: $0$ 不是特征根 (即 $q \neq 0$)<br>  $k=1$: $0$ 是单特征根 (即 $q=0, p \neq 0$)<br>  $k=2$: $0$ 是重特征根 (即 $q=0, p = 0$)                                                              |
| **2. 指数函数乘以多项式** $e^{\alpha x} P_m(x)$ (m次多项式)                                             | $$y^* = x^k e^{\alpha x} Q_m(x)$$<br> ($Q_m(x)$ 为 $m$ 次待定多项式)<br> **$k$ 为 $\mathbf{\alpha}$ 作为特征方程根的重数**：<br>  $k=0$: $\alpha$ 不是特征根<br>  $k=1$: $\alpha$ 是单特征根<br>  $k=2$: $\alpha$是重特征根                                                                                  |
| **3. 三角函数乘以多项式** $P_l(x)\cos\omega x + Q_n(x)\sin\omega x$ <br> ($P_l, Q_n$ 为 $l, n$ 次多项式) | $$y^* = x^k [R_m(x)\cos\omega x + S_m(x)\sin\omega x]$$<br> ($m = \max\{l,n\}$，<br>$R_m, S_m$ 为 $m$ 次待定多项式)<br>**$k$ 为 $\mathbf{\pm i\omega}$ 作为特征方程根的重数**：<br>  $k=0$: $\pm i\omega$ 不是特征根<br>  $k=1$: $\pm i\omega$ 是单特征根 (二阶方程复根只能是单根)                                  |
| **4. 指数函数乘以三角函数和多项式** $e^{\alpha x}[P_l(x)\cos\omega x + Q_n(x)\sin\omega x]$              | $$y^* = x^k e^{\alpha x}[R_m(x)\cos\omega x + S_m(x)\sin\omega x]$$<br> ($m = \max\{l,n\}$，<br>$R_m, S_m$ 为 $m$ 次待定多项式)<br>**$k$ 为 $\mathbf{\alpha \pm i\omega}$ 作为特征方程根的重数**：<br>  $k=0$: $\alpha \pm i\omega$ 不是特征根<br>  $k=1$: $\alpha \pm i\omega$ 是单特征根 (二阶方程复根只能是单根) |

> [!note] 解题步骤
> *   **第一步：确定方程的系数 ($a, b$)。**
>    *   如果题目已给出，则直接使用。
>    *   如果像本题一样通过齐次通解隐含，则根据齐次通解的形式，反推特征根，再由特征根写出特征方程，从而确定系数 $a, b$。
> *   **第二步：求出齐次方程的通解 $Y$。**
>    *   如果像本题一样已给出，则直接使用。
>    *   如果未给出，则根据第一步确定的 $a, b$，写出特征方程并求根，从而得到 $Y$。
> *   **第三步：利用待定系数法求非齐次方程的特解 $y^*$。**
>    *   **关键：** 根据非齐次项 $f(x)$ 的形式，严格按照表格中的规则设取 $y^*$ 的形式（包括 $x^k$ 修正因子和待定多项式 $Q_m(x)$）。
>    *   计算 $y^*$ 的一阶、二阶导数。
>    *   代入原非齐次方程，并比较系数，解出待定系数。
> *   **第四步：写出非齐次方程的通解 $y = Y + y^*$。**
>    *   此时通解中含有任意常数 $C_1, C_2$。
> *   **第五步：利用初始条件确定 $C_1, C_2$。**
>    *   求出 $y(x)$ 的导数 $y'(x)$。
>    *   将初始条件 $y(x_0)=y_0$ 和 $y'(x_0)=y_1$ 代入 $y(x)$ 和 $y'(x)$。
>    *   解一个关于 $C_1, C_2$ 的二元一次方程组。
> *   **第六步：将求得的 $C_1, C_2$ 代回通解，得到最终特解。**

### **Part 3：高阶（$n>2$）常系数齐次线性微分方程**
这类方程是二阶常系数齐次方程的自然推广，解法也是类似。
*   **标准形式**：$y^{(n)} + a_1 y^{(n-1)} + \dots + a_{n-1} y' + a_n y = 0$（$a_1, \dots, a_n$ 为常数）。
*   **求解步骤**：
    1.  **写出特征方程**：
        $$ r^n + a_1 r^{n-1} + \dots + a_n = 0 $$
    2.  **求解特征根**：找到所有 $n$ 个特征根（可能有重根或复根）。
    3.  **根据根的类型和重数，写出对应的通解项，并将所有通解项相加**：

| 特征根类型                        | 对应通解项                                                                                                           |
| :---------------------------- | :--------------------------------------------------------------------------------------------------------------- |
| 单实根 $r$                      | $C e^{rx}$                                                                                                      |
| $k$ 重实根 $r$                  | $(C_1 + C_2 x + \dots + C_k x^{k-1})e^{rx}$                                                                     |
| 单复根对 $\alpha \pm i\beta$     | $e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x)$                                                               |
| $k$ 重复根对 $\alpha \pm i\beta$ | $e^{\alpha x}[(C_1 + C_2 x + \dots + C_k x^{k-1})\cos\beta x + (D_1 + D_2 x + \dots + D_k x^{k-1})\sin\beta x]$ |

### **Part 4：欧拉方程（变系数线性方程的特例）**
欧拉方程是唯一一类考研常考的**变系数**线性微分方程，它可以通过变量替换转化为常系数线性微分方程。

*   **标准形式**：$x^n y^{(n)} + a_1 x^{n-1} y^{(n-1)} + \dots + a_{n-1} x y' + a_n y = f(x)$
    *   **特点**：每一项的 $x$ 的幂次与导数的阶数相同。
*   **求解步骤：变量替换法**
    1.  **令 $x = e^t$** （即 $t = \ln x$）。通常假定 $x > 0$。
    2.  **转化导数**：利用复合函数求导法则，将对 $x$ 的导数转化为对 $t$ 的导数。
        *   $xy' = \frac{dy}{dt}$
        *   $x^2 y'' = \frac{d^2 y}{dt^2} - \frac{dy}{dt}$
        *   $x^3 y''' = \frac{d^3 y}{dt^3} - 3\frac{d^2 y}{dt^2} + 2\frac{dy}{dt}$ (通常考到二阶或三阶)
    3.  **求解转化后的常系数线性微分方程**：将原方程转化为以 $t$ 为自变量的常系数线性微分方程，并按照上述方法求解。
    4.  **回代变量**：将 $t = \ln x$ 代回通解，得到 $y$ 与 $x$ 的通解。

## 15-4 微分方程的几何应用
核心是通过**几何量关系**建立微分方程，常见几何量包括：切线斜率、法线斜率、曲率、面积、体积等。
### 常见应用场景与建模思路
1. **切线 / 法线相关问题**：
    * 切线斜率：曲线上任一点 $(x,y)$ 的切线斜率为 $y'$，切线方程为 $Y - y = y'(X - x)$；
    * 法线斜率：与切线垂直，斜率为 $-\frac{1}{y'}$（$y' \neq 0$）；
    * 建模：根据题目给出的切线 / 法线几何条件（如截距、距离、夹角）列方程。
        示例：求曲线 $y = f(x)$，使曲线上任一点的切线与 $x$ 轴交点的横坐标等于该点横坐标的一半——切线方程令 $Y=0$，得交点 $X = x - \frac{y}{y'}$，由条件 $X = \frac{x}{2}$，得方程 $x - \frac{y}{y'} = \frac{x}{2}$，整理为 $\frac{dy}{dx} = \frac{2y}{x}$。
2. **曲率相关问题**：
* 曲率公式（直角坐标）：$K = \frac{|y''|}{(1 + y'^2)^{\frac{3}{2}}}$；
* 建模：根据题目给出的曲率条件（如曲率为常数、曲率与某量成正比）列方程。

## 15-5 微分方程的物理应用

核心是通过**物理定律**建立微分方程，常见物理场景包括：运动学、热学、电路、力学等。

### 常见应用场景与建模思路

1. **运动学（牛顿第二定律）**：

* 定律：$F = ma$（$F$ 为合外力，$m$ 为质量，$a = y''$ 为加速度，$y$ 为位移）；
* 建模：分析物体受力（如重力、弹力、阻力），列合外力方程，转化为微分方程。

    示例：弹簧振子振动——质量为 $m$ 的物体受弹性力 $-ky$（$k$ 为劲度系数，负号表示与位移反向）和阻力 $-cy'$（$c$ 为阻尼系数），合外力 $F = -ky - cy'$，方程为 $m y'' + c y' + ky = 0$。

1. **热学（牛顿冷却定律）**：

* 定律：物体温度变化率与物体温度 $T$ 与环境温度 $T_0$ 的温差成正比，即 $\frac{dT}{dt} = -k(T - T_0)$（$k > 0$，负号表示温度趋近于环境温度）；
* 建模：直接利用定律列方程，结合初始温度（如 $T(0) = T_1$）求特解。

1. **电路（基尔霍夫定律）**：

* 定律：闭合电路中，电压总和等于各元件电压降总和（如 $L \frac{dI}{dt} + RI + \frac{1}{C} \int I dt = E(t)$，$L$ 为电感，$R$ 为电阻，$C$ 为电容，$I$ 为电流，$E(t)$ 为电源电压）；
* 建模：对电压方程求导（消去积分项），转化为电流 $I$ 的微分方程。
