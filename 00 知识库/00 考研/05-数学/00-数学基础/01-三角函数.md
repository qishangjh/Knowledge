# 三角函数

## 一、常见的三角函数

最常见的三角函数有六个：

| 函数名 | 英文缩写 | 含义（直角三角形） |
|--------|----------|---------------------|
| 正弦   | sin      | 对边 / 斜边         |
| 余弦   | cos      | 邻边 / 斜边         |
| 正切   | tan      | 对边 / 邻边         |
| 余切   | cot      | 邻边 / 对边         |
| 正割   | sec      | 斜边 / 邻边         |
| 余割   | csc      | 斜边 / 对边         |

---

## 二、单位圆与三角函数定义

在单位圆（半径为1的圆）中，三角函数可以推广到**任意角度**（包括大于90°或负角度）：

- 设角θ的终边与单位圆交于点 (x, y)
- 则：
  - sinθ = y
  - cosθ = x
  - tanθ = y/x （x ≠ 0）
  - cotθ = x/y （y ≠ 0）
  - secθ = 1/x （x ≠ 0）
  - cscθ = 1/y （y ≠ 0）

---

## 三、三角函数的基本性质

### 1. 周期性

- sinθ、cosθ 的周期是 **2π**
- tanθ、cotθ 的周期是 **π**

### 2. 奇偶性

- sinθ 是**奇函数**：sin(-θ) = -sinθ
- cosθ 是**偶函数**：cos(-θ) = cosθ
- tanθ 是**奇函数**：tan(-θ) = -tanθ

### 3. 诱导公式（常见）

| 公式 | 说明 |
|------|------|
| sin(π - θ) = sinθ | 补角公式 |
| cos(π - θ) = -cosθ | 补角公式 |
| sin(π/2 - θ) = cosθ | 互余公式 |
| cos(π/2 - θ) = sinθ | 互余公式 |

---

## 四、基本恒等式

### 1. 勾股恒等式

- sin²θ + cos²θ = 1
- 1 + tan²θ = sec²θ
- 1 + cot²θ = csc²θ

### 2. 和差公式
- sin(A ± B) = sinA cosB ± cosA sinB
- cos(A ± B) = cosA cosB ∓ sinA sinB
- tan(A ± B) = (tanA ± tanB) / (1 ∓ tanA tanB)

### 3. 倍角公式

- sin2θ = 2sinθcosθ
- cos2θ = cos²θ - sin²θ = 2cos²θ - 1 = 1 - 2sin²θ
- tan2θ = (2tanθ) / (1 - tan²θ)

## 五、常用角度值（单位：弧度）

| 角度（°） | 弧度 | sinθ | cosθ | tanθ |
|-----------|------|------|------|------|
| 0         | 0    | 0    | 1    | 0    |
| 30        | π/6  | 1/2  | √3/2 | 1/√3 |
| 45        | π/4  | √2/2 | √2/2 | 1    |
| 60        | π/3  | √3/2 | 1/2  | √3   |
| 90        | π/2  | 1    | 0    | 不存在 |

## 六、反三角函数简介

反三角函数用于已知三角函数值求角度，常见有：
- arcsin x（反正弦）
- arccos x（反余弦）
- arctan x（反正切）
它们的定义域和值域是有限的。

---

## 七、应用举例

1. **解三角形**：已知两边一角或两角一边，求其他边角。
2. **周期运动**：如弹簧振动、圆周运动、交流电等。
3. **信号处理**：傅里叶变换中三角函数是基础。
4. **几何与物理**：力的分解与合成、波的叠加等。
