# 韦达定理

韦达定理，也称为根与系数的关系，是关于一元二次方程的根与其系数之间关系的一个重要定理。它是由法国数学家弗朗索瓦·韦达（François Viète）在16世纪末提出的。对于一般形式的一元二次方程 $ax^2 + bx + c = 0$ (其中$a \neq 0$)，如果该方程有两个实数根或复数根 $x_1, x_2$，那么根据韦达定理：

1. 根的和：$x_1 + x_2 = -\frac{b}{a}$
2. 根的积：$x_1 \cdot x_2 = \frac{c}{a}$

这里有几个关键点需要注意：
- 当$a=0$时，方程不再是二次方程而是线性方程，因此韦达定理不适用。
- 韦达定理不仅适用于实数根的情况，同样适用于复数根的情形。
- 通过韦达定理可以直接从给定的方程中得知其解的一些性质，而无需实际求出具体的根值。

此外，韦达定理还可以推广到更高次的多项式方程上，对于一个n次多项式方程 $a_nx^n + a_{n-1}x^{n-1} + … + a_1x + a_0 = 0$ 的所有根 $x_1, x_2, …, x_n$，则有：
- 所有根之和 $x_1+x_2+…+x_n=-\frac{a_{n-1}}{a_n}$
- 所有根两两相乘之和 $\sum_{1\le i<j\le n}x_ix_j=\frac{a_{n-2}}{a_n}$
- 以此类推，直到所有根相乘的积 $x_1x_2…x_n=(-1)^n\frac{a_0}{a_n}$。
