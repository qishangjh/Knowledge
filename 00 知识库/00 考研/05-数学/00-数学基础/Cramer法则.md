# Cramer法则

 :  
【Cramer法则】 n个位置量n个方程的线性方程组$Ax=b$，若$|A|≠0$，则方程组有惟一解$x_i=\frac{D_j}{|A|},j=1,2,3,···,n$ .其中$D_j$是以b代替|A|中第j列所得到的行列式。  
怎么推

 : Cramer法则提供了一种基于行列式求解线性方程组的方法。对于一个形式为$Ax = b$的线性方程组，其中$A$是一个$n \times n$的矩阵，$x$是未知数向量，$b$是常数项向量，如果$A$的行列式$|A| \neq 0$（即$A$是非奇异矩阵），那么根据Cramer法则，这个方程组有唯一解，并且每个未知数$x_i$可以通过以下公式计算：

$$x_i = \frac{D_i}{|A|}, \quad i=1,2,…,n$$

这里，$D_i$是指将矩阵$A$中的第$i$列替换为向量$b$后得到的新矩阵的行列式的值。

## 推导过程

为了理解为什么Cramer法则是正确的，我们可以从以下几个方面入手：

1. **线性方程组与矩阵的关系**：首先认识到，给定的线性方程组$Ax=b$可以被看作是寻找一个向量$x$，使得当它乘以矩阵$A$时，结果正好等于向量$b$。
2. **行列式和逆矩阵**：如果矩阵$A$非奇异（即$|A| \neq 0$），则存在$A^{-1}$，满足$AA^{-1} = A^{-1}A = I$（$I$是单位矩阵）。这意味着我们可以通过两边同时左乘$A^{-1}$来解方程$Ax = b$，从而得到$x = A^{-1}b$。
3. **利用伴随矩阵定义逆矩阵**：矩阵$A$的逆可以用其伴随矩阵$adj(A)$表示为$A^{-1} = \frac{1}{|A|}adj(A)$。伴随矩阵$adj(A)$是由$A$的代数余子式构成的转置矩阵。
4. **代数余子式与$D_i$之间的联系**：特别地，$adj(A)$的每一列实际上就是由$A$中去掉相应行和列后形成的$n-1$阶子矩阵的行列式组成的向量（即$D_i$）经过适当符号调整后的集合。因此，$A^{-1}b$实际上就是将$b$分别与这些调整后的$D_i$相乘后再除以$|A|$的结果。
5. **结论**：因此，当我们用$A^{-1}b$表达$x$时，就得到了Cramer法则的形式，即$x_i = \frac{D_i}{|A|}$，这里的$D_i$正是通过将$b$替换掉$A$中的第$i$列而得到的新矩阵的行列式。

这样，我们就完成了Cramer法则的基本推导。这种方法不仅直观地展示了如何使用行列式来解决问题，同时也揭示了线性代数中矩阵、行列式以及它们之间关系的深刻联系。
